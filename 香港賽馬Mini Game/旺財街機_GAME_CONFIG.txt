## 八、投注方式 (5 種)

| 投注方式 | 挑馬 | 派彩計算 | 備註 |
|---------|------|---------|------|
| 🏆 **獨贏 (WIN)** | 1 匹 | 注碼 × 賠率 | 最基本 |
| 🏅 **位置 (PLACE)** | 1 匹 | 注碼 × (WIN賠率÷3, 最低@1.1) | 前三名即贏 |
| 🔄 **連贏 (Q)** | 2 匹 (不分順序) | 注碼 × (A賠率 × B賠率 × 0.5) | 前兩名即贏 |
| 🎰 **三重彩-順序 (TIERCE)** | 3 匹 (要中順序) | 注碼 × (A × B × C × 0.3) | 最高難度最高回報 |
| 🎰 **三重彩-唔順序 (TRIO)** | 3 匹 (唔使中順序) | 注碼×6 × (A × B × C × 0.05) | 注碼要×6 (3!=6種排列)，賠率較低但較易中 |

> **三重彩兩種玩法**：
> - **順序 (TIERCE)**：要揀中 1-2-3 正確排位。例如 @3×@5×@10 → $100 注碼得 $4,500
> - **唔順序 (TRIO)**：只要 3 匹全中即可。注碼×6=$600 但賠率低得多，得 @7.5

## 九、賠率生成算法

```
綜合能力分 = baseSpeed×25 + stamina×15 + finalSprint×15
           + consistency×10 + energyLevel×10
           + 騎師能力×15 + 練馬師能力×10
勝率 = 該馬能力分 / 全場12匹總和
原始賠率 = 1 / 勝率
最終賠率 = 原始賠率 × 0.82 (馬會利潤)
限制：@1.5 ≤ 賠率 ≤ @99.0
```

## 十四、本地多人模式 (2-4人) — Draft Pick 制

1. 開局設定人數 + 輸入暱稱
2. 每人各 $1,000 啟動資金
3. **Draft Pick 輪流制**：
   - 第 1 場：A → B → C → D 順序下注
   - 第 2 場：D → C → B → A 反轉順序
   - 第 3 場：A → B → C → D (輪轉)
   - 如此類推，類似 NBA Draft Pick
4. **下注公開可見**：所有人嘅選擇都攤開，後面嘅玩家可以睇到前面揀咗咩
5. **策略深度**：先手可以搶到心儀馬匹，後手可以根據對手選擇避開或跟注
6. 所有人下注完 → 比賽開始 → 衝線後同步顯示結果
7. 10 場完畢 → 總排名 + 最終冠軍！

> **設計原意**：公開 + 輪轉令每場都有博弈策略，唔只係靠分析，仲要考慮對手行為
