第一章：遊戲總覽 (Game Overview)
═══════════════════════════════════════════════════════════════════════════

名稱：旺財街機 (Wong Choi Arcade)
類型：像素風 (16-bit) 2D 橫向捲軸賽馬模擬 × 情報分析策略遊戲
入口：Dashboard 第三個 Nav Pill「🕹️ 街機」
模式：單機模式 + 本地多人模式 (1-4 人)

定位：
  不只是一個簡單嘅賽馬動畫 — 而是一個結合 Wong Choi 分析方法論嘅
  「模擬投注策略遊戲」。玩家需要閱讀賽前新聞、分析馬匹數據、
  做出基於情報嘅投注決定，然後觀賞比賽結果。

核心體驗：
  1. 每個「賽日」包含 10 場賽事 (Race 1–10)
  2. 每場有 12 匹馬出賽
  3. 玩家以 $1,000 旺財幣開局
  4. 逐場下注，第 10 場結束後結算總成績
  5. 賽前提供「旺財情報」(新聞、Barrier Trial、騎師連勝等)
  6. 遊戲完結後可選擇「再嚟一日」重置

遊戲模式：

  🎮 單機模式 (Solo Mode)
  ──────────────────────
    - 1 人遊玩，挑戰自己嘅最高紀錄
    - 全部 10 場由你一個人落注
    - 目標：用 $1,000 旺財幣搵到最多錢
    - localStorage 記錄歷史最高得分 + 最佳命中賠率

  👥 本地多人模式 (Local Multiplayer — 2-4 人)
  ─────────────────────────────────────────────
    玩法設計：同一部電腦/手機上輪流落注，睇同一場比賽！

    流程：
      1. 開局設定人數 (2-4 人) + 輸入暱稱 (如「旺財」「福星」)
      2. 每人各自擁有 $1,000 旺財幣啟動資金
      3. 每場比賽前，玩家輪流落注：
         - 畫面顯示「🎯 輪到 [旺財] 落注！」
         - 該玩家選馬 + 輸入注碼
         - 下注後自動切換到下一位玩家
         - 其他玩家嘅選擇會被隱藏 (唔會互相睇到)
      4. 所有人落注完畢 → 比賽開始！
      5. 衝線後同時顯示所有人嘅結果：
         ┌────────────────────────────────────────┐
         │ 🏆 第 3 場賽果                          │
         │ ─────────────────────────────────        │
         │ 旺財 → 精英大師 @2.5 → 第1名 ✅ +$250  │
         │ 福星 → 金槍六十 @4.0 → 第3名 ❌ -$100  │
         │ 阿德 → 靚蝦王  @8.0 → 第2名 ❌ -$50   │
         └────────────────────────────────────────┘
      6. 第 10 場後，總結排名：
         🥇 旺財：$2,350 (總盈利 +$1,350)
         🥈 福星：$1,100 (總盈利 +$100)
         🥉 阿德：$680  (總虧損 -$320)
      7. 可選「再嚟一日」全員重置

    勝負條件：10 場結束後最多旺財幣者贏
    社交元素：衝線時會顯示所有人嘅選擇，製造歡笑 & 爭議！

3.4 模擬賽績生成系統 (Race Record Generator)
────────────────────────────────────────────

  每匹馬會根據其評級自動生成「近六場模擬賽績」：

  生成規則：
    ✦ S/A 級馬：近六場名次從 [1-4] 中隨機，偶爾 [5-7]
      例如：1-2-1-3-1-1 (超級穩定)
    ✦ B 級馬：近六場名次從 [2-6] 中隨機，偶爾 [1] 或 [7-9]
      例如：3-5-2-4-6-3 (有實力但唔穩定)
    ✦ C 級馬：近六場名次從 [4-9] 中隨機，偶爾 [2-3]
      例如：7-4-9-5-8-6 (中游浮沉)
    ✦ D 級馬：近六場名次從 [7-12] 中隨機，偶爾 [4-6]
      例如：11-8-12-9-10-7 (長期包尾)

  每場賽績包含嘅資訊：
    ✦ 場地 (沙田 / 跑馬地)
    ✦ 距離 (1000m - 2400m)
    ✦ 名次 (用圈數字 ①②③...⑫ 顯示)

  統計摘要自動計算：
    ✦ N戰N冠 (勝場數)
    ✦ 勝率 % (冠軍次數 / 總場數)
    ✦ 入三甲率 % (前三名次數 / 總場數)

  賽績與遊戲機制嘅互動：
    ✦ 賽績同馬匹嘅隱藏屬性值直接掛鉤 — 表現好嘅馬真係能力強
    ✦ 但有「寬恕因素」嘅馬，其賽績會故意更差 (模擬「被低估」)
    ✦ 冷門馬信號會標記近績差但有回勇跡象嘅馬
    ✦ 玩家可以透過賽績做出更有根據嘅投注決定

  跨賽日賽績更新 (進階功能)：
    ✦ 多人模式下，每匹馬喺當日賽事嘅結果會被記錄
    ✦ 下一個賽日再抽到同一匹馬時，佢嘅賽績會更新
    ✦ 例如：金槍六十今日跑第 2 → 下次佢嘅賽績最新一場顯示 ②

4.1 賽日結構 (Race Day Structure) — 動態隨機生成
──────────────────────────────────────────────────

  每個「賽日」由 10 場賽事組成。賽事嘅距離、班次同特色
  唔再固定，而係根據場地 (日間沙田 / 夜間跑馬地) 動態隨機生成。

  🌙 跑馬地賽日 (Happy Valley — 夜間模式)
  ───────────────────────────────────────
  跑馬地為內圈小馬場，以低班賽事為主，冷門爆發率較高。

  距離池：1000m, 1200m, 1650m, 1800m, 2200m
    (跑馬地冇 1400m/1600m，改用獨有嘅 1650m)

  班次分佈 (10 場隨機抽)：
    ┌──────────┬──────────────────────────────────────┐
    │ 班次      │ 出現機率 / 場數                        │
    ├──────────┼──────────────────────────────────────┤
    │ 第五班    │ ~30% (約 3 場) — 新馬/低班，冷門天堂   │
    │ 第四班    │ ~30% (約 3 場) — 普通賽事              │
    │ 第三班    │ ~25% (約 2-3 場)                      │
    │ 第二班    │ ~10% (約 1 場) — 跑馬地難得嘅高班      │
    │ 第一班    │ ~5%  (約 0-1 場) — 極罕見，壓軸用途    │
    │ 國際賽    │ 0%   — 跑馬地唔會舉辦國際級賽事         │
    └──────────┴──────────────────────────────────────┘
  特色：整體賠率分佈更分散、冷門馬勝率稍高 (模擬真實)

  ☀️ 沙田賽日 (Sha Tin — 日間模式)
  ──────────────────────────────────
  沙田為國際級大馬場，高班賽事集中，大熱門勝率較高。

  距離池：1000m, 1200m, 1400m, 1600m, 1800m, 2000m, 2400m
    (沙田跑道長，支持全距離範圍)

  班次分佈 (10 場隨機抽)：
    ┌──────────┬──────────────────────────────────────┐
    │ 班次      │ 出現機率 / 場數                        │
    ├──────────┼──────────────────────────────────────┤
    │ 第五班    │ ~10% (約 1 場)                        │
    │ 第四班    │ ~15% (約 1-2 場)                      │
    │ 第三班    │ ~25% (約 2-3 場)                      │
    │ 第二班    │ ~25% (約 2-3 場) — 中堅賽事            │
    │ 第一班    │ ~15% (約 1-2 場) — 高班精英            │
    │ 國際賽 🏆 │ ~10% (約 1 場) — 壓軸大賽，R10 限定   │
    └──────────┴──────────────────────────────────────┘
  特色：賠率較緊密、大熱門勝率高、D 級馬基本冇機會

  📋 賽日生成規則：
    ✦ R1 一定係最低班 (第五班或第四班) — 模擬真實暖場賽
    ✦ R10 一定係當日最高班 — 壓軸賽事
    ✦ 中間 R2-R9 按機率池隨機分配，但整體走勢由低班到高班
    ✦ 距離從距離池中隨機抽取，確保每個賽日距離唔會太重複
    ✦ 每場嘅「特色標籤」亦隨機生成 (新馬賽 / 讓賽 / 盃賽 等)

  班次對馬匹質素同隨機性嘅影響：
    ✦ 第五班 → 能力分佈最闊 (0.5-2.5), consistency 0.4-0.7, 冷門率 ~30%
      → 隨機性最大！什麼馬都有機會贏，分析較難但賠率誨人
    ✦ 第四班 → 能力分佈中 (1.0-3.0), consistency 0.5-0.75, 冷門率 ~22%
      → 隨機性仍然很高，但開始有跡可尋
    ✦ 第三班 → 正常分佈 (1.5-3.2), consistency 0.6-0.85, 冷門率 ~12%
      → 分析開始發揮作用，賠率和實力比較掛鉤
    ✦ 第二班 → 較窄分佈 (2.0-3.4), consistency 0.7-0.9, 冷門率 ~6%
      → 實力說了算，冷門事件少見
    ✦ 第一班 → 精英分佈 (2.5-3.5), consistency 0.8-0.95, 冷門率 ~3%
      → 大熱門幾乎穩贏，偏差極小
    ✦ 國際賽 → 極窄分佈 (3.0-3.5), consistency 0.85-0.98, 冷門率 ~2%
      → 全部都係頂級馬，分析最有意義但競爭最激烈

4.2 馬匹狀態機與物理引擎 (Horse Physics — 擴充版)
──────────────────────────────────────────────────

  每匹馬擁有 10 項隱藏屬性值 (Stats Map)：

  ┌─────────────────┬───────────────────────────────────────────┐
  │ 屬性名稱          │ 說明                                      │
  ├─────────────────┼───────────────────────────────────────────┤
  │ baseSpeed        │ 基礎速度 (1.0-3.5)                        │
  │ stamina          │ 耐力值 (40-100)                           │
  │ burstChance      │ 爆發機率 (3%-30%)                         │
  │ burstPower       │ 爆發力度 (1.3x-2.8x)                     │
  │ finalSprint      │ 末段衝刺 (0.3-1.8)                        │
  │ consistency      │ 穩定性 (0.5-1.0)，數值越高速度波動越小     │
  │ gateSpeed        │ 出閘速度 (0.5-1.5)，影響起步搶位           │
  │ trackPreference  │ 場地偏好 (good/yielding/soft)              │
  │ distanceFit      │ 路程適性 (short/mid/long)                  │
  │ energyLevel      │ EEM 能量值 (50-100)，模擬 Wong Choi EEM   │
  └─────────────────┴───────────────────────────────────────────┘

  額外屬性 (從 Wong Choi 分析系統提取)：

  ┌─────────────────┬───────────────────────────────────────────┐
  │ 屬性              │ 說明                                      │
  ├─────────────────┼───────────────────────────────────────────┤
  │ runningStyle    │ 跑法：領放/居前/居中/後上                   │
  │ barrierDraw     │ 檔位 (1-12)，影響彎位走位同出閘優勢       │
  │ weight          │ 磅數 (110-133磅)，磅越重速度越慢          │
  │ isDebut         │ 首出馬標記 (true/false)，屬性隱藏         │
  │ gearChange      │ 配備變動 (眼罩/舌繫帶 等)，影響特定屬性    │
  └─────────────────┴───────────────────────────────────────────┘

  Wong Choi 概念融合：

  🔬 段速法醫 (Sectional Speed)：
    - finalSprint 參數即為段速法醫嘅遊戲化版本
    - 高 finalSprint 值 = L600 段速優秀 = 末段衝刺能力強
    - 數值會在賽前情報中以「段速報告」形式暗示畀玩家

  ⚡ EEM 能量管理 (Energy Expenditure Model)：
    - energyLevel 直接模擬 Wong Choi 嘅 EEM 系統
    - 連續出戰嘅馬匹 energyLevel 會降低 (模擬疲勞)
    - 休息後復出嘅馬匹 energyLevel 會回復滿值
    - 影響 stamina 消耗速度：低能量 → stamina 損耗加快

  📋 寬恕檔案 (Forgiveness Archive)：
    - 系統會隨機為某些馬匹生成「寬恕因素」
    - 有寬恕因素嘅馬匹，其「顯示賠率」會偏高（因為近績差）
    - 但實際隱藏能力值會被上調 (模擬「被低估」)
    - 賽前情報會暗示：「上仗受阻，實力未盡展現」

  🔍 冷門馬信號 (Underhorse Signal)：
    - 每場有 0-2 匹馬可能被標記為冷門馬候選
    - 三級信號系統：🟢 輕度 / 🟡 中度 / 🔴 強烈
    - 信號越強，該馬觸發「爆冷」事件嘅機率越大
    - 爆冷觸發時，該馬會在最後直路突然加速 (finalSprint ×2.5)

4.2b Wong Choi 進階遊戲機制 (深度玩法)
────────────────────────────────────────────

  以下機制全部從 Wong Choi 分析系統提取，令遊戲變得更像真實賽馬分析：

  🏇 跑法系統 (Running Style)：
    每匹馬有一個固定跑法，影響嘅不僅係速度，仲係全場走位同策略：
    ┌─────────┬────────────────────────────────────────────┐
    │ 跑法     │ 特徵                                         │
    ├─────────┼────────────────────────────────────────────┤
    │ 領放     │ 起步最快，前段領先但後段 stamina 消耗大   │
    │         │ 優勢：短途賽 (1000-1200m) + 好地           │
    │         │ 劇情：領先全程但末段可能被追過               │
    ├─────────┼────────────────────────────────────────────┤
    │ 居前     │ 穩健型，跟住領放馬但留力                 │
    │         │ 優勢：全距離適用，最平均                │
    │         │ 劇情：穩步前進但沒有大反彈               │
    ├─────────┼────────────────────────────────────────────┤
    │ 居中     │ 中段待機，最後直路發力                 │
    │         │ 優勢：中距離賽 (1400-1800m)               │
    │         │ 劇情：末段有機反超但也可能追唔到           │
    ├─────────┼────────────────────────────────────────────┤
    │ 後上     │ 起步最慢，但 finalSprint 最強          │
    │         │ 優勢：長途賽 (2000m+) + 慣性強步速       │
    │         │ 劇情：末段大反彈追過所有，最刺激嘅跑法   │
    └─────────┴────────────────────────────────────────────┘
    跑法影響動畫：領放馬起步衝前，後上馬起步在尾但末段電射追上！

  🎰 檔位效應 (Barrier Draw Effect)：
    每場賽事 12 匹馬會隨機分配檔位 (1-12 號)：
    - 檔 1-4 (內欄)：彎位優勢 +5%，但容易被外馬過擋
    - 檔 5-8 (中檔)：無特別加成，最平均
    - 檔 9-12 (外檔)：跑外彎懲罰 -5%，但過彎時唔會被阻
    - 戴不同場地嘅偏差：跑馬地內欄偏差明顮，沙田較平均
    - 賠率卡片會顯示檔位號，經驗豐富嘅玩家可以利用

  ⚖️ 磅數讓磅系統 (Handicap Weight)：
    模擬香港賠馬會嘅讓磅制度：
    - S/A 級馬背 128-133 磅 (重磅懲罰)
    - B 級馬背 120-127 磅 (標準)
    - C 級馬背 115-121 磅 (輕磅優勢)
    - D 級馬背 110-116 磅 (最輕磅)
    - 每增加 1 磅 = baseSpeed -0.5%
    - 讓磅制令低班馬有更多機會，增加策略性
    - 下注卡片會顯示磅數，玩家可以判斷蟒不蟒賠率

  🌟 首出馬系統 (Debut Runners)：
    每場有 5% 機率出現 1 匹「首出馬」：
    - 賠績欄顯示「無賠績 — 首次出賽」
    - Wong Choi 指標部分隱藏，顯示「❓」
    - 但賽前情報可能會透露試閘表現
    - 實際能力從 S 到 D 都有可能，係最大嘅未知數！
    - 賠率通常較高 (@10-@40)，因為市場唯有觀望
    - 策略：如果情報提示試閘優秀，可能偏值很高

  🔧 配備變動情報 (Gear Changes)：
    某些馬匹會隨機獲得「配備變動」標記，出現在情報中：
    - 👁 首次上眼罩 (Blinkers On)：集中力 +15%，consistency +10%
    - 📢 首次上舌繫帶 (Tongue Tie)：呪儀通暢度 +8%，stamina +5%
    - 👣 換了新馬蹄鐵 (New Shoes)：跑地抑地力 +5%
    - 配備變動可以係正面或負面 (即刳罩 ≠ 一定好)
    - 賠前情報額外提示：「🔧 金槍六十今場首次上眼罩作戰」

  🏆 成就解鎖系統 (Achievement System)：
    隨遊戲進度解鎖更多內容，增加重玩性：

    青銅成就 (1 個賽日內可達成)：
      ✦ 「初出茅廬」— 完成第一個賽日
      ✦ 「大熱門獲勝」— 買中 @3.0 以下大熱門
      ✦ 「冷門多博」— 投注了一匹 @30+ 嘅馬

    白銀成就 (需要多個賽日)：
      ✦ 「精算繩數」— 累計盈利超過 $5,000
      ✦ 「冷門獵人」— 成功命中 3 匹 @20+ 嘅冷門馬
      ✦ 「連勝穩將」— 喺同一賽日連中 4 場

    金色成就 (傳奇級)：
      ✦ 「精英大師」— 單一賽日盈利超過 $10,000
      ✦ 「賽馬大亨」— 累計完成 10 個賽日
      ✦ 「不敗之神」— 整個賽日 10 場全中
      ✦ 「票房大屏殺」— 命中了 @50+ 嘅超級冷門

    解鎖獎勵：
      ✦ 報告 — 彈出像素風「成就解鎖！」動畫 + chiptune 音效

4.3 賽事物理流程 (Race Physics Flow)

  ┌─────────────────────────────────────────────────────────┐
  │ Phase 1: 出閘 (Gate Start)                    0-10%    │
  │   → gateSpeed 決定起步排序                              │

  │   → burstChance 隨機觸發短暫加速 (持續 1-3 秒)          │

  │   → finalSprint 參數啟動                                │

  │   → 冷門馬「爆冷」事件可能觸發                           │
  │   → 騎師特殊技能啟動 (例如莫雷拉末段 +15%)              │
  │   → 戲劇性末段反超前！                                  │
  └─────────────────────────────────────────────────────────┘

4.3a 步速視覺化系統 (Pace Display)
────────────────────────────────────

  比賽過程中畫面頂部會有即時步速指示器，讓玩家看到全場步速：

  ┌────────────────────────────────────────┐
  │ 步速：🔥 快  [==========━━━━━] 65%  │
  │ 領放: #3 金槍六十                      │
  │ 後上群: #7 步步友 · #12 嘉應高昇      │
  └────────────────────────────────────────┘

  步速對賽果嘅影響：
    ✦ 🔥 快步速：領放馬有利，但末段 stamina 消耗更快
      → 後上馬有更大機會追上！
    ✦ ❄️ 慢步速：領放馬眍住步速，stamina 保存得更好
      → 後上馬追起嘰較輛
    ✦ 🎯 均速：居前/居中馬最有利，穩定發揮
  步速由領放馬嘅 gateSpeed + baseSpeed 決定，每場都唔同！

  │                      │         │ 3-5 個身位                │

  │ 🔀 走位受阻          │ ~8%     │ 某匹馬被前方馬匹遮擋       │

  │                      │         │ 之後能力正常恢復，但已失   │
  │                      │         │ 去位置。成為下場「寬恕」   │
  │                      │         │ 候選。                     │
  │                      │         │ 動畫：馬匹短暫卡住 + ❗   │

4.4 動態賠率系統 (Dynamic Odds)
───────────────────────────────

  賠率生成算法：
    1. 綜合能力分 = baseSpeed×25 + stamina×15 + finalSprint×15
                  + consistency×10 + energyLevel×10
                  + 騎師能力×15 + 練馬師能力×10
    2. 勝率 = 該馬能力分 / 全場 12 匹馬能力分總和
    3. 原始賠率 = 1 / 勝率
    4. 馬會利潤率調整 = 原始賠率 × 0.82
    5. 最終賠率 = 四捨五入至小數後一位
    6. 最低賠率限制 = @1.5 (大熱門保護)
    7. 最高賠率限制 = @99.0

  賠率範圍分佈 (12 匹馬)：
    - 1-2 匹：@1.5 - @4.0 (大熱)
    - 2-3 匹：@4.0 - @10.0 (次熱)
    - 3-4 匹：@10.0 - @25.0 (中游)
    - 3-4 匹：@25.0 - @99.0 (冷門)

4.5 派彩與資金系統 (Payout & Balance)
──────────────────────────────────────

  起始資金：$1,000 旺財幣
  最低注碼：$10
  最高注碼：當前餘額嘅 50% (防止 all-in)

  🎰 投注方式 (Bet Types)：
  ┌───────────┬────────────────┬───────────────────────────────┐
  │ 投注方式    │ 挑馬方式         │ 派彩計算                          │
  ├───────────┼────────────────┼───────────────────────────────┤
  │ 🏆 独贏    │ 揀 1 匹馬贏    │ 注碼 × 賠率                       │
  │ (WIN)      │                │ 例：$100 × @3.5 = $350            │
  ├───────────┼────────────────┼───────────────────────────────┤
  │ 🏅 位置    │ 揀 1 匹馬      │ 注碼 × 位置賠率 (約 WIN 賠率    │
  │ (PLACE)    │ 前三名即算贏  │ ÷ 3，但不低於 @1.1)             │
  │            │                │ 例：@3.5 → 位置 @1.3            │
  ├───────────┼────────────────┼───────────────────────────────┤
  │ 🔄 連贏    │ 揀 2 匹馬      │ 注碼 × (A馬賠率 × B馬賠率 × 0.5) │
  │ (Q連贏)    │ (不分順序)    │ 前兩名即算贏                     │
  │            │                │ 例：@3.5 + @8.0 = $100 × 14.0    │
  ├───────────┼────────────────┼───────────────────────────────┤
  │ 🎰 三重彩  │ 揀 3 匹馬      │ 注碼 × (A × B × C × 0.3)       │
  │ (TIERCE)   │ (要中顺序)    │ 須按正確順序 1-2-3               │
  │            │                │ 例：@3×@5×@10 = $100 × 45.0    │
  │            │                │ 高難度但回報極豐厚！           │
  └───────────┴────────────────┴───────────────────────────────┘

  投注 UI 流程：
    1. 揀投注方式 (点擊同一排嘅 4 個像素按鈕)
    2. 揀馬 (點擊馬匹卡片，Q/三重彩需要點多匹)
    3. 輸入注碼
    4. 確認下注 → 開跑！

  多人模式特別規則：
    ✦ 每位玩家可以釀不同投注方式
    ✦ Q/三重彩更強的風險回報，增加玩家間嘅競爭策略
    ✦ 結算畫面顯示每人用咗種投注方式 (WIN/PLACE/Q/T)

  派彩邏輯：
    - 買中：餘額 += 注碼 × 對應賠率
    - 買唔中：注碼歸零
    - 如果餘額跌至 $0 → 「破產」畫面 → 可選擇「借貸」$500 重新開始
      (但會記錄「借貸次數」影響最終評分)

  賽日結算 (10 場後)：
    - 顯示總成績：贏/輸場數、總盈利/虧損、最高賠率命中
    - 評級系統：
      ├ S 級 (傳奇)：盈利 > $5,000
      ├ A 級 (高手)：盈利 $2,000 - $5,000
      ├ B 級 (穩健)：盈利 $500 - $2,000
      ├ C 級 (普通)：盈利 $0 - $500
      ├ D 級 (輸家)：虧損 $0 - $500
      └ F 級 (破產)：虧損 > $500 或曾借貸
