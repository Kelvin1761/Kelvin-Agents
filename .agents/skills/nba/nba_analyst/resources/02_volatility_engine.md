# 波動性引擎與情境調整 (Volatility Engine & Contextual Adjustments)

## Step 2 Part 1: 基礎波動率計算

### 計算流程
針對每位候選球員,在內部推理中逐步計算:

**1a. 分鐘標準化 (Minute-Adjusted Projections)**
若球員 L5 平均上場時間波動 > 5 分鐘(即最高場 vs 最低場差 > 10 分鐘),必須執行 Per-36 標準化:
```
Per-36 該場數據 = (原始數據 / 實際上場分鐘) × 36
```
用於識別「因上場時間減少而數據偏低」的假右波動。若 Per-36 與原始 AVG 偏差 > 15%,在報告中標註 `⚠️ 上場時間波動大`。

> 注意:盤口線仍用原始數據計算(因為投注對象係實際數字),但 Per-36 用於輔助判斷球員係「真的狀態下滑」定係「只係打少咗」。

**1b. L3 加權機制 (Recency Weighting)**
除了計算標準 AVG 外,同時計算加權平均值:
```
Weighted AVG = (L1-L3 × 1.5 + L4-L7 × 1.0 + L8-L10 × 0.7) / (3×1.5 + 4×1.0 + 3×0.7)
```
若 Weighted AVG 與標準 AVG 差異 > 2.0,在報告中標註趨勢方向(↑ 近期上升 / ↓ 近期下滑)。

**1c. 基礎計算**
1. **AVG** (平均值) = Sum(L10) / 10
2. **MED** (中位數) = L10 排序後嘅中間值
3. **SD** (標準差) = √(Σ(xi - AVG)² / N)
4. **CoV** (變異數係數) = SD / AVG

### CoV 分級

| CoV 範圍 | 標籤 | 說明 | 適用組合 |
|:---------|:-----|:-----|:---------|
| < 0.15 | 🛡️ 極度穩定 | 像公務員,最適合做穩膽 | 穩膽首選 |
| 0.15 - 0.25 | ✅ 穩定 | 可做穩膽組合 | 穩膽/價值 |
| 0.25 - 0.35 | ➖ 一般波動 | 需配合正面情境 | 價值/高賠 |
| > 0.35 | 🎲 神經刀 | 避免穩膽,僅適合高賠/爆發 | 高賠/避開 |

## Step 2 Part 2: 情境調整加減分表

| Pace-Up (慢遇快) | +1.0 | 節奏加速 |
| Pace-Down (快遇慢) | -1.0 | 節奏減速 |
| 近期有不穩定因素 (場外負面新聞、被教練點名) | -2.0 | 無法用數據衡量嘅風險 |

| 隊友傷缺紅利 (同位置主力Out,吸收球權) | +1.5 | 角色提升 |
| Sharp Money 順勢買大 | +1.0 | 聰明錢訊號 |
| 莊家誘餌/聰明錢買小 | -2.0 | 謹慎規避 |
| 休息日紅利 (3 日休息) | +1.0 | 額外休息紅利 |
| 超額休息紅利 (4+ 日休息) | +1.5 | 體能全滿加成 |
| H2H 歷史明確趨勢 (≥3 場 H2H 數據) | +1.0 / -1.0 | 若 H2H 表現穩定超/不達盤口 → 加/減分 |
| H2H 數據不足 (<3 場) | 不適用 | 跳過 H2H 調整 |

### 調整值計算規則
1. 累加所有適用嘅調整值。
2. **調整後預期值 = AVG + 總調整值**。
3. 若總調整值為負數且幅度 > 5.0,該球員應被強烈警戒或剔除。

## Step 3 Part 1: 盤口雙線生成

### 穩膽線 (Banker Line)
```
穩膽線 = (AVG - 0.5 × SD) + 情境調整值
```
- 目標命中率 ≥ 80%
- 從 L10 中計算:有幾場超過此線?
- 穩膽線賠率必須 ≥ 1.80

### 價值線 (Value Line)
```
價值線 = MED + 正面加成項總和
```
- 目標命中率 ≥ 75%
- 價值線賠率必須 ≥ 2.50

### AMC 評估 (平均超過幅度)
- 計算球員 L10 中超過盤口線嘅場次平均超標幅度
- **AMC 0.5-1.0**:⚠️ 汗水單 (Sweaty Bet),空間極低
- **AMC 3.0+**:🚀 碾壓優勢,高信心

## Step 3 Part 2: +EV 篩選

### 計算公式
```
預估真實勝率 = Min(95%, L10 命中率 ± 情境調整影響)
莊家隱含勝率 = 1 / 賠率
Edge = 預估真實勝率 - 莊家隱含勝率
```

### Edge 評級
| Edge | 評級 |
|------|------|
| > 15% | 💎 核心高價值 |
| 10-15% | ✅ 有價值 |
| 5-10% | ➖ 邊緣 |
| < 5% | ❌ 不推薦 |

### Under 偵測系統
若球員符合以下條件,直接考慮買 Under:
- 高波動 (CoV > 0.35) **且** 遇上精英防守對位
- 遇到 Vegas Trap 誘餌(盤口低但 Over 賠率異常高)
- 球隊客場飛航 B2B
- 近期有嚴重負面新聞

---

## 無即時賠率處理
若無法獲取即時賠率,使用以下美式基準估算:
- 標準盤口 Over/Under:賠率 1.90
- Alt Lines (降維):根據跳階距離調整
