## 3. 反惰性掃描

### 3.1 省略語搜尋（遇到任何一個即 FAIL）
搜尋以下字串：
- `...`
- `[同上]`
- `[略]`
- `[參見組合`
- `[完整數據見組合`
- `[FILL]`
- `[FILL:`

### 3.2 深度一致性
- [ ] 組合 1B 嘅每 Leg 字數 ≥ 組合 1A 每 Leg 字數嘅 80%
- [ ] 組合 2 嘅每 Leg 字數 ≥ 組合 1A 每 Leg 字數嘅 80%
- [ ] 組合 3 嘅每 Leg 字數 ≥ 組合 1A 每 Leg 字數嘅 80%

### 3.3 L10 數組校驗
- [ ] 每個 L10 逐場數組包含恰好 10 個數字
- [ ] 同一球員喺唔同組合出現時，L10 數組相同（盤口線唔同但原始數據應一致）
- [ ] 兩位唔同球員嘅 L10 數組唔完全相同（防止幻覺模式 DATA-002）

## 4. 數學一致性校驗

| 公式 | 容差 |
|------|------|
| 隱含勝率 = (1/賠率) × 100 | ≤ 1.5% |
| Edge = 預估勝率 - 隱含勝率 | ≤ 0.6% |
| CoV = SD / 均值 | ≤ 0.02 |

## 5. 跨場一致性（≥2 場時適用）

- [ ] 前 2 場 vs 後 2 場嘅平均 Leg 分析長度差距 ≤ 40%
- [ ] 所有場次使用相同嘅欄位結構
- [ ] 所有場次嘅組合命名一致（1A/1B/2/3）

## 6. Bet365 合規

- [ ] 所有盤口數字符合 Bet365 嚴格選項規則（Strict Built-in Matrix）
- [ ] 得分盤口：14.5, 19.5, 24.5, 29.5 等 Milestone 線（嚴禁 22.5 等自創線）
- [ ] 籃板/助攻盤口：整數或 .5 結尾
