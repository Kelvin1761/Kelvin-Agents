### Step 5: 狀態與穩定性指數 [增強版]

**5.1** 近 10 仗入三甲比例:>50% = 鐵腳;>70% = 超級鐵腳
**5.2** `STABILITY_RANK` 全場比較 + 波動類型判斷
**5.3** Step 3 新鮮度風險 → 降低穩定性信心一級
**5.4 場地專家辨識:** 同場同程 ≥5 勝 → `[場地專家]`。詳見 `06_rating_engine.md` Step 14.3。
**5.5 條件型馬辨識:** 同場同程 ≥2 勝但非首本大起大落 → `[條件型馬]`。若今仗符合勝出條件 → 穩定性 ❌ 覆蓋為 ➖。
**5.6 沉睡專家回師豁免:** 近 4 仗 0 入五甲但近 10 仗有同場同程入三甲 → 穩定性強制 ➖，標記 `[沉睡專家回師]`。
  - 升級:配合 Step 6.5 正面訊號可觸發寬恕反彈。失效:同場同程最近 3 次全未入五甲。
**5.7 質新馬加速進步豁免:** ≤5場 + 近 3 仗連續改善 ≥3 位(如 8→6→3) → 穩定性升格 ✅。Rising Star:≤4歲 + 連續改善 + 同騎師連騎 ≥2 → 微調 +半級。

### Step 6: 隱藏變數與速勢

**引用 Step 0:** `PACE_TYPE` 判斷跑法受惠/受損,`BIAS_DIRECTION` 判斷逆偏差。
- 配備/負磅變動(> 5 磅)
- **大幅配備變動升級:** ≥2 項配備變動 + Tier 1/2 騎師 → 練馬師訊號強制 ✅。單項變動走 Step 14.2B 微調。互斥:若練馬師訊號已因配備觸發(Size Gear Reset 等),此處不重複。
- 末段分析:慢步速但末段追近 = 隱藏強馬;快步速仍能貫注 = 確認強馬
- 標記「追勢有餘 (Eye-catching)」

**引擎-步速錯配偵測:**

| 錯配情境 | 觸發條件 | 效果 |
|:---|:---|:---|
| Type B 龜速困局 | Type B + 龜速/偏慢 + LEADER ≤ 2 | 情境 ❌ |
| Type A 搶位消耗 | Type A + 自殺式 + LEADER ≥ 3 | 形勢 ❌ |
| Type A 龜速壟斷 | Type A + 龜速 + 唯一前領 | 情境 ✅✅ + 微調升級 |
| Type B 自殺式紅利 | Type B + 自殺式/真實快 + 排 1-6 檔 | 情境加分 |

**極密出練馬師信心訊號:** 上賽 ≤14日 + (狀態與穩定性 ❌ 或 質新馬 ➖ 有入三甲) → 騎練訊號 ✅（核心維度），可佐證積極部署意圖。

**場地狀態影響 (Going Impact):**
- Going = Good-to-Yielding/Yielding + 無濕地紀錄 → ⚠️ 場地適性未知
- Going/AWT 互動:草地 Going 不影響 AWT 賽事

**距離轉換風險矩陣:**

| 距離變動 | 風險等級 | 處理 |
|:---|:---|:---|
| ±0-100m | 忽略 | 無調整 |
| +200m | 低 | 查 Sire AWD + 近績 |
| +400m+ | ⚠️ 高 | `[增程大躍進]`。豁免:同等/更長績優；大勝；Stamina Sire |
| -200m~-400m | 中 | 查縮步幅能力 + 「縮程賺級數」正面因素 |

**引擎類型 × 距離轉換修正:**

| 距離變動 | Type A | Type B | Type C |
|:---|:---|:---|:---|
| 縮程 -200m | ⚠️ 中高 | ✅ 低 | ➖ 中 |
| 增程 +200m | ➖ 低 | ⚠️ 中 | ✅ 低 |
| 增程 +400m | ➖ 中 | ❌ 高 | ⚠️ 中 |
| 縮程 -400m | ❌ 高 | ➖ 中低 | ➖ 中 |

**縮程級數紅利:** 縮程 ≥200m + 長路程評分/班次 ≥ 今仗 → `✅ [縮程賺級數]`。失效:目標短程 ≥3 次全未入三甲。Type A 需額外驗證段速模式。

**巔峰久休豁免:** 35-56日 + 休息前 4 仗 ≥2 勝 / ≥3 入三甲 / 明確上升軌 → 馬匹健康 / 新鮮感 ❌ 降為 ➖。標記 `[巔峰休養]`。

> [!IMPORTANT]
> **🔴 路程否決權:** 以下觸發硬性封頂 C+：
> 1. 極端縮程 ≥30% 且目標場地路程無入三甲
> 2. 場地類型轉換(泥↔草）且 ≤1 次近路程入三甲 → 額外降一級
> 3. 豁免:同路程同場已勝出 / 練馬師配備測試(改 D 封頂)

**[實績覆蓋血統]:** ≥2 勝或 ≥3 上名 → 實績完全覆蓋 Sire AWD。嚴禁以血統降級。Sire AWD 僅用於首出/輕驗馬。
