# 場地邏輯模組:跑馬地草地 (Happy Valley Turf)

> 本模組僅適用於跑馬地賽事。
> 03-06 入面嘅跑馬地條件規則(Step 5 HV 專家矩陣、Step 6 HV 隱藏變數、Step 14.2B/14.3 谷草微調)自動生效。

---

## 核心特性 — Valley Bias

| 因素 | 影響 | 解釋 |
|:---|:---|:---|
| 短直路 (~280m) | 🔴 極高 | 僅沙田嘅 60%,後追馬發力空間嚴重不足 |
| 急彎(半徑小) | 🔴 極高 | 外疊馬幾何蝕位遠大於沙田 |
| 內檔偏差 | 🔴 極高 | 1-4 檔慳盡彎位距離,優勢極大 |
| 前置偏差 | 🟡 高 | 領放/前驅唔使喺短直路追趕,天然優勢 |

## 跑道配置 × 影響

| 配置 | 影響 |
|:---|:---|
| A | 外檔仍有少許空間,相對最公平 |
| B | 檔位重要性上升,前置馬更佔優 |
| C / C+3 | 極度狹窄。外檔(≥8) + 後追型 = 幾乎判死刑(所有距離適用,不限 1000m) |

## 距離特性

| 距離 | 特性 |
|:---|:---|
| 1000m | 起步即入彎,外檔(≥8)致命。最利內檔前速馬 |
| 1200m | 過 2 個急彎。檔位影響高 |
| 1650m | 跑馬地最常見距離。1 完整圈 + 額外直路。戰術性最高 |
| 1800m | 氣量要求高,但短直路仍然限制後追馬 |
| 2200m | 接近 1.5 圈。極度考驗氣量,短直路仍然限制後追馬 |

## Step 0 修正
- 跑馬地場景下,2 匹搶位馬嘅效果 ≈ 沙田 3 匹(急彎加劇搶位消耗)
- 「領放主導階梯」評估時,Jockey Authority 權重應提高(谷草步速控制更依賴騎師技術)

## Step 14 互動(現有規則確認 + 擴展)
- 跑馬地贏馬數據 1.5× 權重 ✅ (已有)
- 五班谷草 1-3 檔加分 ✅ (已有)
- 谷草磅差封頂 ✅ (已有, SIP-HV17 分級化)
- **擴展:** C/C+3 賽道 + 外檔(≥8) + 後追型 = 觸發「致命死檔」微調降級(所有距離適用,不限 1000m)

## [SIP-HV16] B 賽道 + 慢步速組合規則 (HV-B Track Slow Pace Bias)

> [!IMPORTANT]
> 當 B 賽道 + 預計步速為「慢」或「中等偏慢」+ 向後移欄時,以下規則**強制生效**:

| 條件 | 效果 |
|:---|:---|
| 檔 1-5 + 前置/跟前型 | 情境維度自動升為 ✅(內欄前置紅利極大化)+ Step 14.2B 步速形勢微調升級 |
| 檔 1-5 + 後追型 | 情境維度 ➖(內欄有位但後追短直路仍受限) |
| 檔 6-8 + 任何跑法 | 無特殊調整(維持現有規則) |
| 檔 ≥9 + 後追型 | 觸發「致命死檔」微調降級 -1 級(慢步速 + 外疊 + 短直路 = 三重不利) |
| 檔 ≥9 + 前置型 | 觸發 ⚠️ 風險警示(前速可搶位但仍覆蓋額外距離) |

> **與 SIP-HV10 互動:** 若為 1800m/2200m,致命死檔門檻仍按 SIP-HV10 放寬至 ≥12 檔,但 SIP-HV16 的檔 ≥9 懲罰降格為 ⚠️ 排序降半級(不觸發完整致命死檔)。
> **來源:** 2026-03-25 HV R1 開心三多(A+, 10檔, 後追型)未入前4 vs 富喜來(D, 5檔, 前置型)冠軍 9.4x。全日 9 場內欄馬系統性受惠。

## [SIP-HV22] B 賽道慢步速內欄超級加權 (HV-B Inner Draw Super Boost)

> [!IMPORTANT]
> **2026-03-25 盲測驗證來源:** R1 實際前4名全部為檔1-5(富喜來檔5冠軍、夢照發檔4亞軍、管之友檔1季軍、架勢奇爸檔2殿軍)。SIP-HV16 僅給予情境維度 ✅ 不足以反映內欄壟斷性。

當 B 賽道 + 慢步速/中等偏慢 + 向後移欄時,**除 SIP-HV16 情境維度外**,額外觸發以下排序加權:

| 條件 | 效果 |
|:---|:---|
| 檔 1-5 + 任何跑法 | Top 4 排序中 ✅ 有效數 **+0.5**(無論評級等第,內欄馬在同等或相近評級間排序優先) |
| 檔 1-3 + 任何跑法 | Top 4 排序中 ✅ 有效數 **+1.0**(黃金內欄,結構優勢極大) |

> **注意:** 此加權為**排序用途**(同等第之間嘅先後次序),不改變基礎評級等第(A-仍是A-)。但可影響 Pick 1-4 嘅次序。
> **與 SIP-HV16 互動:** SIP-HV16 嘅情境✅ 和 SIP-HV22 嘅排序加權可疊加。
> **與 SIP-HV21 互動:** SIP-HV21 嘅谷草磅差折扣和 SIP-HV22 嘅排序加權獨立計算,兩者可抵銷。

## [SIP-HV23] HV 長途後追 + 內欄慳位豁免 (HV Long-Distance Closer Inner Exemption)

> [!IMPORTANT]
> **2026-03-25 盲測驗證來源:** R1 管之友(檔1、後追型、135lb)被引擎判為情境 ❌(後追+慢步速=三重不利),但實際跑季軍。HV 2200m 嘅入彎次數多(接近1.5圈),內欄後追馬嘅慳位效應累積極大。

| 條件 | 規則 |
|:---|:---|
| HV ≥1800m + 檔 1-3 + 後追型 | 情境維度從 **❌ 降格為 ➖**(長途慳位效應部分補償後追劣勢) |
| HV ≥1800m + 檔 4-5 + 後追型 | 維持現有 SIP-HV16 判定(➖) |

> **原理:** HV 2200m 接近 1.5 圈,每過一個急彎內欄馬慳約 2-3 個馬位。3-4 個彎位累計慳 6-12 個馬位 = 相當可觀嘅體力儲備。檔 1-3 嘅後追馬可以「貼欄等彎、直路衝」。

## [SIP-HV24] 頂磅 + 好檔位補償 (Top Weight Good Draw Compensation)

> **2026-03-25 盲測驗證來源:** R1 管之友(135lb頂磅、檔1)被 SIP-RR17(頂磅壓力)+ SIP-HV17(谷草磅差封頂)雙重懲罰至 C-,但實際跑季軍。

| 條件 | 規則 |
|:---|:---|
| HV 賽事 + 負磅 ≥133lb + 檔 1-3 | SIP-RR17 頂磅壓力降級**減半**(降半級 → 降四分一級,即排序降格而非等第降格) |
| HV 賽事 + 負磅 ≥133lb + 檔 4-5 | SIP-RR17 頂磅壓力降級維持**原有力度** |

> **原理:** 頂磅馬配好檔位(1-3)時,內欄慳位效應可部分抵銷磅差劣勢。但僅限檔1-3(極內欄),檔4-5 仍按原規則。
> **與 SIP-HV17 互動:** SIP-HV17 谷草磅差封頂不受此規則影響(封頂仍然有效),此規則僅減緩 SIP-RR17 嘅降級力度。

## [SIP-HV24b] 頂磅 + 外檔加罰 (Top Weight Outer Draw Penalty)

> [!WARNING]
> **2026-03-25 盲測驗證來源:** R2 佳登(135lb、檔7、A-)在 HV 1650m B賽道未入前4。同場至高心得(119lb、檔1、B-)跑殿軍。頂磅+非內欄 = 每一步都比對手消耗更多,無法慳位。

| 條件 | 規則 |
|:---|:---|
| HV 賽事 + 負磅 ≥133lb + 檔 ≥6 | 在 SIP-RR17 降級基礎上額外**再降半級**(頂磅無法慳位 = 劣勢放大) |
| HV 賽事 + 谷草磅差 ≥15lb + 檔 ≥8 | **硬性封頂 B+**(不可維持 A- 或以上) |

> **原理:** 頂磅馬最需要內欄慳位來節省體力。檔≥6 意味著無法沿欄節省腳程,頂磅劣勢被結構性放大。檔≥8 + 大磅差(≥15lb) 更是致命組合。
> **與 SIP-HV24 互動:** SIP-HV24(好檔補償)和 SIP-HV24b(外檔加罰)為同一邏輯嘅正反兩面,不會同時觸發(因檔位互斥)。

## [SIP-HV25] HV 短途磅差折扣 + 內欄騎師升級 (HV Sprint Weight Discount)

> **2026-03-25 盲測驗證來源:** R3 開心孖寶(135lb、檔2、艾道拿Tier 1、C-)跑季軍。1200m短途磅差影響遠低於中長途(70秒 vs 140秒),且HV 1200m 2個急彎令內欄優勢極大。

| 條件 | 規則 |
|:---|:---|
| HV ≤1200m 短途 | SIP-HV17 谷草磅差門檻**放寬 +3lb**(例如原本18lb中度 → 21lb才算中度) |
| HV ≤1200m + 檔 1-3 + Tier 1/2 騎師 | 級數維度❌ **降格為 ➖**(內欄+好騎師可補償頂磅劣勢) |

> **原理:** 1200m賽事只跑約70秒,頂磅馬嘅負擔累積時間僅為2200m嘅一半。加上HV 1200m有2個極急彎,內欄馬每個彎慳2-3個馬位。Tier 1/2騎師能精確掌控起步+彎位走位,進一步放大內欄優勢。
> **與 SIP-HV24 互動:** SIP-HV25嘅短途磅差折扣與SIP-HV24嘅好檔補償可疊加(兩者均受惠於內欄位置)。

## [SIP-HV26] 沙田轉跑馬地路程豁免 (ST→HV First-Time Transfer Exemption)

> **2026-03-25 盲測驗證來源:** 3匹「首跑HV」馬全部冠軍!R4 爆出美麗(布文,檔1,🏆), R6 馬達(布文,檔7,🏆15x), R8 凌登(周俊樂,檔12,🏆)。ST→HV轉場並非劣勢。

| 條件 | 規則 |
|:---|:---|
| 首跑HV(只有ST紀錄) | 路程維度標記為 **➖偏正面** + **排序加權+0.5**(ST→HV轉場屬正面訊號) |
| 首跑HV + ST同程有入三甲紀錄 | 路程維度標記為 **✅**(ST能力可直接轉移,等同有HV紀錄) |
| 首跑HV + 檔≥10 | 路程維度標記為 **➖**(中性,外檔抵銷轉場利好) |

> **原理:** HV與ST均為香港賽馬場,馬匹定期在兩場之間轉換。ST→HV轉場並非結構性劣勢——許多馬匹反而因HV較小場地而受惠。練馬師選擇轉場往往有積極意圖(見 `07b_trainer_signals.md` [轉場試驗] 訊號)。
> **SIP-HV15 互動:** S-閘門嘅「場地經驗確認」規則仍然有效——首跑HV不可評S-。但路程維度不再因首跑HV而自動❌。
