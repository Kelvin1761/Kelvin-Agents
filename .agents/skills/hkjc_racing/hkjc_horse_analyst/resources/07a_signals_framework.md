# 練馬師與騎師出擊訊號 — 框架 (Framework)

本文件定義練馬師/騎師訊號系統嘅使用規則。
具體嘅練馬師訊號同騎師 profile 分別喺 `07b_trainer_signals.md` 同 `07c_jockey_profiles.md`。

**條件式讀取指引:**
- Wong Choi 會喺數據包提供 `[ACTIVE_TRAINERS]` 同 `[ACTIVE_JOCKEYS]`
- 讀取 07b 時,只需關注 `[ACTIVE_TRAINERS]` 中列出嘅練馬師訊號
- 讀取 07c 時,只需關注 `[ACTIVE_JOCKEYS]` 中列出嘅騎師 profile
- 不在名單中嘅練馬師/騎師使用以下 Fallback 規則

## ⚠️ Fallback 規則(騎師/練馬師不在列表時)

以上列表為常見模式,不包含所有騎師/練馬師。若騎師或練馬師不在列表中:
- **騎師分級:** 按當季勝場排名分類 — Top-3 = 頂級(適用溢價校正旗標檢查);Top-10 = 主力(標準評估);10 名以外 = 次選(後追型配搭需警惕)
- **練馬師訊號:** 不在列表 = 無訊號 (➖ Neutral)。不可因練馬師不在列表而自動給予 ❌。
- **見習騎師通則:** 任何見習騎師(減磅 ≥3lb)配後追型馬匹 = 自動觸發 `[騎師誤配]`(同黃寶妮/鍾易禮邏輯一致)

- **騎師反差配訊號 (Jockey Contrast Signal):** 若馬匹**近 3 仗 ≥1 仗跑入前五名**(有走勢),但今仗配 10 名以外的次選騎師,不應因騎師排名而自動觸發降級或增加 ❌。改為觸發 `[反差配 (Contrast Deploy)]` 標記,視為練馬師有信心的中性至正面訊號:
  - **前速型馬 + 次選騎師** → `[反差配]` = ➖ Neutral(前速馬不太依賴騎師技術)
  - **後追型馬 + 次選騎師** → `[騎師誤配]` ❌ 維持(後追依賴騎師發力點判斷)
  - **前速型馬 + 次選騎師 + 輕磅 (≤118lb)** → `[反差配+輕磅]` = 視為半核心練馬師訊號 ✅(練馬師擺明配輕磅冷門局)
  - **觸發「反差配」後,覆蓋 `[騎師誤配]` 扣分** — 兩者互斥,不可同時生效。
