**Step 14.2B:微調因素(在 14.2 評級後、14.3 覆蓋前檢查):**

> [!IMPORTANT]
> 以下微調分兩個通道獨立計算：
> - **通道 A（現有微調）：** 以下因素未納入 8 維度,但可將 14.2 評級上調或下調 **最多一級**。多個因素同時觸發時,升降互相抵消,最終淨調整不超過一級。

**Step 14.2C: D 級強制冷門掃描 (D-Grade Mandatory Longshot Scrub):**

> [!IMPORTANT]
> 此步驟在 14.2B 完成後、14.3 強制再覆蓋前**強制**執行。
> 目的是避免將具備單一強力因素的負面馬匹一棍子打進冷門 D 級而被完全忽視。負面馬爆冷出的機率比一般預期高得多。

若馬匹經 14.2 + 14.2B 後評級為 **D 或 C-/C** ,強制反問是否存在以下任一「全場稀有的強力正面訊號」:

| 強力正面訊號 | 門檻 |
|:---|:---|
| 急劇降磅 | 今仗負磅比上仗輕 ≥15 磅 |
| 第一次更換最佳騎師 | 潘頓/布文/莫雷拉等高手 |
| 回師首本路程場地 | 歷史內檔 ≥2 次同程入三甲 |
| 激烈配備變動 | 首次戴眼罩/重戴正面配備且有歷史證據 |
| 超級降班馬 | Step 8.1 觸發超級降班旗標 |
| 路程場地專家 | 同場同程 ≥2 次勝出紀錄 |
| **首出馬強力試閘** | 試閘 Top-2 且留力 + 練馬師季內首出勝率 ≥ 15% |
| **首出馬內檔部署** | 首出馬 + 1-3 檔 + 一線騎師 (Top-10) |

若以上存在 ≥1 個強力正面訊號:
- D 級馬 → **強制升級至 C-**
- C-/C 級馬 → **強制升級至 B-**
- 加入標記 **`🏂冷門候選待確認 (Longshot Candidate)`**
- 必須在「第四部分 分析盲區 Step 6」中列出該馬的強力因素

> 若以上全部訊號均不存在 → D 級維持不變。

**Step 14.2E: B+ 級升級掃描 (B-Plus Upgrade Scan):**

> 此步驟在 14.2D 完成後**強制**執行。2026-03-25 HV 中有 4 匹 B+ 馬勝出或跑入前二,但現有掃描僅覆蓋 D/C-/C/B- 級,B+ 成為盲區。

若馬匹經 14.2B 後評級為 **B+**,檢查是否**同時**滿足以下 ≥2 項條件:

| 條件 | 門檻 |
|:---|:---|
| HV 場地經驗 | 同場(跑馬地)同程(±100m)≥1 次入三甲紀錄 |
| 近仗走勢 | 近 3 仗 ≥1 仗跑入前五 |
| 檔位優勢 | 排 1-6 檔 |
| 輕磅紅利 | 負磅 ≤ 全場中位數 |
| 練馬師/騎師正面訊號 | 練馬師訊號為 ✅ 或騎師為 Top-10 |

若 ≥2 項滿足 → **升級至 A-**,標記 `[B+ HV 專家升級 (B+ HV Specialist)]`。
若僅 1 項 → 維持 B+ 但標記為 `🐴⚡ B+ 潛力馬`(冷門馬訊號觸發)。
- **來源:** 2026-03-25 HV R4 爆出美麗(B+→🏆3.9x)、R6 馬達(B+→🏆15x)、R8 凌登(B+→🏆5.9x)、R8 瑪瑙(B+→亞軍)


**Step 14.2F: 大熱崩潰壓力測試 (Favourite Collapse Stress Test):**

> [!WARNING]
> **來源:** 2026-04-06 ST R4 威武年代(S, @4.7→6th)、R5 堅先生(A+, @3.0→6th)、R1 閃電小子(A-, @1.5→2nd)。短賠大熱門+風險因素組合 = 高崩潰率。

當引擎首選（Pick 1）滿足以下**全部條件**時，強制執行壓力測試：

| 條件 | 門檻 |
|:---|:---|
| 評級 | ≥ A |
| 預計獨贏賠率 | ≤ 5.0 |
| 以下任一風險 | (a) 負頂磅(≥133lb)；(b) 檔位 ≤5(1400m以上賽事)；(c) 休後復出 >60日 |

**效果：**
1. **強制寫出「崩潰情境」段落**（在 Verdict 前）：
   - **目標馬：** [馬名] | 評級：[X] | 預計賠率：[Y]
   - **崩潰誘因：** [頂磅/內檔/休後...]
   - **崩潰機率評估：** [低/中/高]
   - **若崩潰，最受惠馬：** [馬名（第二選）]
   - **資金分配調整：** 若崩潰機率 ≥ 中 → 首選注碼削減 30%，分配至第二/三選
2. **第二選最低評級門檻：** Pick 2 必須 ≥ B+ 評級。若 Pick 2 < B+，強制重新審視 T4 排列。
3. **崩潰機率判定標準：**
   - **低：** 僅 1 項風險因素 且馬匹有證明紀錄克服該因素
   - **中：** 2 項風險因素 或 1 項無克服紀錄
   - **高：** ≥3 項風險因素

**不觸發：** 首選預計賠率 > 5.0；首選評級 < A。

**Step 14.2G: 二班以上後追馬加分 (Class 2+ Deep Closer Bonus):**

> [!NOTE]
> **來源：** 2026-04-06 ST R10 綠族無限(@44, 走位 10→10→8→1，未入 T4)。高班 1400m 賽事中證明嘅後追馬被系統性低估。

**觸發條件：**

**效果：**

**不觸發：** 班次 ≥ 第三班；距離 ≠ 1400m；近 5 仗後追成功率 <2 次。
