# 初出馬及進口馬分析指引 (HKJC V1.0)

> **條件載入:** 當 WorkCard 標記 `career_tag = DEBUT` 或 `IMPORTED_DEBUT` 時載入此指引。
> 已有正式賽績的馬匹一律使用標準流程；不再使用 `EARLY_CAREER` tag。

## 適用條件
- `career_tag = DEBUT` (香港零出賽 + 本地馬)
- `career_tag = IMPORTED_DEBUT` (進口馬香港首出)

## ⚠️ 重要：禁止錯誤分類
- 只有香港正式賽績 0 場先可以寫「初出」/「首出」/「無往績」
- 已有任何正式賽績的馬匹**嚴禁**使用初出馬模板或評級封頂語言
- ✅ 正確表述：已有賽績馬用標準近績、段速、形勢、騎練模板
- ❌ 錯誤表述：「休出後」/「復出後」

---

## A. DEBUT 本地馬 (HK Career = 0, 非進口)

> 使用同一套維度矩陣，但判定依據有所調整。
> **Rating Cap: A。** 初出馬仍由同一套 7 維矩陣 ✅ counts 計算，但 final rating 最高 A，不可進入 A+ / S-tier。

### 維度特殊處理

| 維度 | 類型 | 處理 |
|:---|:---|:---|
| 穩定性 | 半核心 | **預設 ➖ Neutral，但可用備戰穩定性升/跌**。初出馬「無正式賽績」本身不等於差；✅ 必須引用操練連貫、試閘次數、readiness、體重/健康穩定；❌ 必須引用操練中斷、trial反覆、健康或體重風險 |
| 段速質量 | 核心 | 正式賽事段速未知；改用 Sire 早熟度 / Sire AWD 距離投射 + 試閘速度/末段姿態 + 晨操 readiness 作替代證據。缺 Sire 或 trial/trackwork evidence 不可憑空 ✅ |
| 形勢與走位 | 半核心 | 只用檔位數據、今日預計位置、試閘跑法/出閘反應；不得用步速作此維度主要加減分 |
| 騎練訊號 | 核心 | 維持核心。初出馬嘅強晨操訊號（賽日騎師親操、加壓曲線、配備意圖、兩次或以上試閘）優先 route 入騎練訊號；有強訊號時不可只寫「待正式賽績驗證」 |
| 級數優勢 | 輔助 | 一般 ➖ (同場皆為初出) 或 ❌ (面對有經驗且有勝績對手)；只有 Sire + trial + trainer evidence 極強先可正面 |
| 場地 | 輔助 | 一律 ➖（無場地紀錄） |
| 賽績線 | 輔助 | 一律 N/A / 不計入（無正式對手追蹤數據），不可當 ❌ |

### A Cap 觸發條件

初出馬只有同時具備以下三類證據，先可最高評至 A。A cap 由評級封頂處理，**不靠硬壓 stability 達成**：
1. **Sire/血統:** 早熟度或 Sire AWD / 子嗣距離投射吻合今仗。
2. **試閘/晨操:** 試閘 Top-3 且留力/輕策，或晨操 readiness 高、加壓曲線清楚、末段姿態強。
3. **騎練部署:** 練馬師初出 KPI、騎師親操、一線騎師、配備意圖其中至少一項強訊號。

只滿足兩類最高 A-；只滿足一類最高 B+；缺 Sire 或 trial/trackwork 關鍵資料時不可憑空給 A/A-。

## B. IMPORTED_DEBUT 進口馬 (HK Career = 0, 有海外賽績)

### 特殊處理
- **穩定性**：可參考海外賽績序列 → 有數據基礎，不限 ➖
- **段速**：可參考海外段速（如有）→ 但需考慮場地差異，信心度打七折
- **級數**：海外賽績可作為級數參考基準（海外 Group 賽經驗 = 正面因素）
- **場地**：參考海外場地表現，但考慮香港場地差異
- **Rating Cap**：同標準馬匹（因為有海外賽績支持，唔設封頂）

---

## C. 已有賽績馬 (HK Career ≥ 1)

使用標準分析框架。即使只有 1-2 場正式賽績，都要按正常模板完整分析：
1. 穩定性：以已跑賽績、頭馬距離、走勢可靠性判斷，不可寫「無正式賽績」。
2. 段速：以正式賽 L400 / 全段速剖面 / 完成時間偏差作主證據。
3. 晨操：只作狀態及部署補充，不能把已有賽績馬改寫成初出馬。
4. core_logic 不可使用「初出馬」「首出」「評級封頂」等字眼。
