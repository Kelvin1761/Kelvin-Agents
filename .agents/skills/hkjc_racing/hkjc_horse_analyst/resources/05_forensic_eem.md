### Step 10: 段速法醫分析

**絕對禁止只看名次。**

> [!CAUTION]
> **[SIP-P1a] 全段速剖面強制引用 (Full Sectional Mandate):** 段速法醫分析**嚴禁只引用 L400 單一數字**。每匹馬必須引用 Facts.md「📊 全段速剖面」表格中嘅 **S1/S2/S3...各段時間** 及 **Δ1/Δ2/Δ3...偏差值**，對每段速度逐一解讀。若 Analyst 僅引用 L400 而忽略前段/中段數據 → **QG-CHECK 自動失敗**。

**10.1** 提取近仗全段速剖面（優先）及 L600/L400 段速
**10.2 段速修正(定性判斷,非精確計算):**

| 修正因素 | 條件 | 修正方向 |
|:---|:---|:---|
| 三疊望空 | 全程外疊無遮擋 | 實際能力 **優於** 末段時間所示 |
| 受困 | 直路/彎位受阻 | 實際能力 **明顯優於** 末段時間(未盡全力) |
| 逆偏差 | 內欄利但走外 | 實際能力 **優於** 名次所示 |
| 順偏差 | 享受利好偏差 | 實際能力 **弱於** 末段時間所示(打折扣) |
| 領放 | 前 400m 用力領放 | 末段放慢屬正常,不扣分 |

**⚠️ 保守修正原則:** 段速修正嘅結論係「實際能力優於/弱於時間所示」,屬**定性方向判斷**,不等於「大幅升級」。特別係「三疊望空」同時出現於段速修正 (Step 10) 同 EEM 潛力 (Step 11) 時,段速修正應保守處理 — 僅確認「能力不弱於所示」,避免同 EEM 維度雙重膨脹。

**10.3** 近 3 仗修正後末段趨勢:逐仗改善 = `上升軌`;逐仗退步 = `衰退中`;持平 = `穩定`

**10.4 [SIP-P1b] 段速形態一致性分析 (Sectional Shape Consistency):**

> [!IMPORTANT]
> 從 Facts.md 全段速剖面中提取近 3 仗嘅「形態」分類（漸進加速/快開慢收/均速型/慢開快收/波動型），判斷形態一致性：

| 近 3 仗形態一致性 | 判定 | 意義 |
|:---|:---|:---|
| 3 仗相同形態 | `高一致性 ✅` | 跑法高度可預測,分析置信度高 |
| 2 仗相同 + 1 仗不同 | `中等一致性 ➖` | 基本可預測,但有變數 |
| 3 仗各不同 | `低一致性 ⚠️` | 跑法不穩定,段速數據參考價值降低 |

**形態一致性傳導:** 高一致性 → 段速法醫結論可信度加分;低一致性 → 段速法醫結論需降低權重,不可作為核心判定依據。

**10.5 [SIP-P1c] 前段-後段因果分析 (S1-to-L400 Causality):**

必須對每匹馬嘅近 3 仗,分析前段 Δ 同後段 Δ 嘅因果關係：

| 前段 Δ | 後段 Δ | 判讀 | 引擎質量訊號 |
|:---|:---|:---|:---|
| 大正數（慢開）| 大負數（快收）| 「儲力爆發」型 | 正常 — 需扣除儲力效應才能評估真實末段能力 |
| 大負數（快開）| 大正數（慢收）| 「過度消耗」型 | ⚠️ 負面 — 前段用力過度導致後段衰退 |
| 大負數（快開）| 大負數（快收）| 「全程強勢」型 | ✅✅ 極度正面 — 快開仍能快收 = 引擎質素極高 |
| 大正數（慢開）| 大正數（慢收）| 「全程不及班次」型 | ❌ 負面 — 即使儲力亦無餘勁 |

**量化門檻:** Δ 偏差 ≥ |0.5s| 為「大」偏差;Δ 偏差 < |0.3s| 為「輕微」偏差。

**10.6 [SIP-P1d] 頭馬距離趨勢驗證 (Margin Trend Verification):**

必須引用 Facts.md 嘅「頭馬距離趨勢」數據,同段速法醫結論進行交叉驗證：

| 段速趨勢 | 頭馬距離趨勢 | 交叉驗證結論 |
|:---|:---|:---|
| L400 上升軌 ✅ | 📈 收窄中 | ✅✅ 高度驗證 — 段速進步 + 距離收窄 = 真實進步 |
| L400 上升軌 ✅ | 📉 擴大中 | ⚠️ 矛盾信號 — 段速快但整體落後更多 = 可能係全場水平提升 |
| L400 衰退中 ❌ | 📈 收窄中 | ⚠️ 正面修正 — 段速字面退步但實際競爭力接近 = 班次因素 |
| L400 衰退中 ❌ | 📉 擴大中 | ❌❌ 高度負面 — 段速退步 + 距離拉大 = 確認衰退 |

**[SIP-ED5] 引擎類型段速解讀框架 (Engine Type Sectional Interpretation Framework):**

段速質量維度嘅 ✅/❌ 判定必須先確認引擎類型 (Step 2.6),再以該類型嘅預期段速模式作為比較基準:

| 引擎類型 | 預期段速模式 | L400 偏慢判定 | L400 偏快判定 |
|:---|:---|:---|:---|
| **Type A (前領均速)** | 均勻衰減,L400 不會特別快 | ➖ **正常** — 不應因此判 ❌ | ✅ **異常正面** — 前領型仍有末段加速 = 實力深不見底 |
| **Type B (末段爆發)** | L400 應為最快段,顯著快於全場平均 | ❌ **異常負面** — 標記 `⚠️ 引擎失效 (Engine Misfire)` | ✅ **正常** — 確認引擎正常運作 |
| **Type C (持續衝刺)** | 段速穩定,各段偏差小 | ➖ **正常** — 若整體段速穩定即可 | ✅ 若穩定中仍有加速 = 正面 |
| **Type A/B (混合)** | 無固定預期 | 按實際表現判斷 | 按實際表現判斷 |

**「引擎失效」(Engine Misfire) 規則:**
- 觸發條件:Type B 馬匹 + 近 3 仗 L400 持續排全場後半段(≥ 50% 分位)
- 效果:段速質量維度強制判為 ❌ Weak,且在分析中標記 `⚠️ 引擎失效 — 末段爆發型 L400 持續偏慢,核心武器失靈`
- 原理:Type B 嘅存在意義就係 L400 爆發力。若 L400 持續偏慢 = 引擎本身有問題,而不只係「一仗爬頺」

**→ 段速修正判斷傳入 Step 11 作 EEM 基準**

### Step 11: 能量消耗模型 (EEM 2.0)

**幾何與動力學修正 (Geometry & Dynamics):**
定性評估馬匹的走位軌跡消耗,必須結合步速與賽道物理特性。

> [!IMPORTANT]
> **EEM 場地幾何與豁免條款(必須優先檢查):**
> 1. **1000m 直路賽:** 無彎道,外疊不構成幾何蝕位,此類賽事 **自動關閉** EEM 多跑距離懲罰。
> 2. **末段抽出 (Late Sweep):** 前半程守 1-2 疊,僅最後 400m 抽出 3-4 疊衝刺。這是正常最佳跑法,**不可**視為高消耗。
> 3. **泥地防沙機制 (AWT Kickback):** 沙田泥地 (AWT) 走 3-wide 望空 **不一定** 是高消耗。若為避開前列食沙 (Kickback) 而刻意走外疊走順步韻,其消耗低於被困內欄食沙。
> 4. **沙田 1200m vs 1400m 起步點:** ST 1200m 排外檔 (10-14) 切入內欄的「早段耗力 (Early Burn)」極高;但 ST 1400m 有長直路緩衝,外檔耗力顯著較低。
> 5. **谷草 C/C+3 窄欄陷阱:** 跑馬地 C 或 C+3 跑道極度狹窄,排外檔 (10-12) 除非是絕對級數超班(如 `[超級降班馬]` 或曾有 `[逆境破格]` 紀錄),否則極易陷入致命的 3-wide 全程高消耗。

**基礎走位消耗參考:**

| 走位 | 消耗等級 | 物理意義 |
|:---|:---|:---|
| 1-wide 貼欄 / 輕鬆放頭 | 低消耗 | 基準,幾何距離最短,無阻力 |
| 2-wide 二疊 / 甚至有遮擋 | 中低消耗 | 略有額外半徑距離,體力影響輕微 |
| 轉彎/全程 3-wide 望空 | 高消耗 | 承受全程向心力與多跑距離,體力透支 |
| 早段火燒 (Early Burn) | 高消耗 | 為搶前/切入內欄而在前 400m 耗盡體力(即使最終守低消耗位) |
| 轉彎/全程 4-wide | 極高消耗 | 幾何蝕位極其嚴重,幾乎不可能交出後勁 |

**EEM 步速緩衝判定 (Pace Buffer):**
- 若為 `龜速 (Crawl)` + 3-wide:降格為 **「中等消耗」**(慢步速下外疊較不費力,反而免受擠阻)。
- 若為 `極快 (Suicidal)` + 3-wide:升格為 **「極高消耗」**(毀滅性體力透支)。
- **[慢步速擋位紅利 (Slow Pace Positional Bias)]:** 當步速預期為 `中等偏慢 (Moderate-to-Slow)` 或 `龜速 (Crawl)` 時,排 1-4 檔的馬匹自動獲得「位置保證紅利」。在 B/C+3 等移欄賽道,此紅利極度放大,即使馬匹級數稍遜(如 C 級),也必須在對比外檔馬時給予情境加分。
- **[慢步速互斥條件]:** 若一匹馬已因「慢步速擋位紅利」(前置好位)獲得情境加分,不可同時因 Step 6「慢步速但末段追近 = 隱藏強馬」再獲額外加分。兩者描述不同受惠機制(前者利前置,後者利後追),同一匹馬只能觸發其中一個。
- **[快步速輕磅紅利 (Fast Pace Light Weight Bonus)]:** 當步速預期為 `快 (Genuine-to-Fast)` 或 `自殺式 (Suicidal)` 時,負磅 ≤120lb 的後追/中疊馬獲得額外「能量補償紅利」。前方重磅馬 (>130lb) 在快步速下崩潰機率極高,輕磅馬的體力優勢在末段會被成倍放大。在谷草短直路 (B/C+3 賽道) 此效應更加顯著。
- **[快步速崩潰特例 (Suicide Pace Collapse)]:** 在極快步速下,最大的受惠者不一定是「大外引發後勁」的馬匹。若有能在內欄 (1-2疊) 留守前中列、沿途慳盡腳程的馬匹,它們在對手崩潰後能從內疊竄上,這是最高效益的「地利+步速」受惠者。

**[SIP-ED6] 引擎類型 EEM 消耗修正 (Engine Type EEM Consumption Modifier):**

EEM 消耗判定必須考慮馬匹引擎類型 (Step 2.6) 對體力基線嘅影響:

| 引擎類型 | EEM 消耗修正 | 原理 |
|:---|:---|:---|
| **Type A + 2-wide 無遮擋** | 消耗等級 **提高一檔**(中低→中,中→高) | 前領型全程做 pace = 體力基線已高,額外外疊消耗嘅邊際影響更大 |
| **Type B + 3-wide 望空** | 消耗等級 **降低半檔**(若步速為中等或快) | 末段爆發型喺中後段外疊跑可能係刻意策略(避開塞車),若仍能收出強勁末段 = EEM 逆境破格信號更強 |
| **Type A + 龜速 + 獨步前領** | 走位消耗 **忽略** | 獨放頭 + 龜速 = 基本零消耗,無論走幾 wide 都不構成體力問題 |
| **Type B + 龜速 + 後追** | 消耗等級 **提高一檔** | 後追型喺龜速中被迫居後過遠,末段需要覆蓋更多距離才能追到,體力消耗反而更大 |

**與現有豁免條款嘅互動:**
- EEM 步速緩衝 (Pace Buffer):龜速 + 3-wide 降格為中等消耗 → 若馬匹為 Type A 且為獨放頭,進一步降格為「低消耗」
- 慢步速擋位紅利:若馬匹為 Type A + 排 1-4 檔 + 龜速 → 已經通過 SIP-ED1 匹配矩陣獲得情境加分,EEM 消耗修正可進一步加強該加分效果

**EEM 潛力觸發 (矩陣判斷基準):**
- **✅ Strong (四大觸發條件):**
  1. `[逆境破格 The Monster]`:近仗經歷幾何「真高消耗」(三/四疊望空)卻依然勝出或上名 → 證明其級數遠超物理限制。
  2. `[超級反彈 The Rebounder]`:近仗經歷幾何「真高消耗」僅敗或輸近,且今仗客觀形勢大幅改善(如檔位 14→1,或步速逆轉) → 釋放隱藏實力。**若配合當日利內欄偏差 (Rail Bias),此反彈效應倍增。**
  3. `[隱形消耗反彈 The Rhythm Rebound]`:近仗看似幾何「低消耗」(1-2疊),但遭遇嚴重的**隱形消耗**(如:嚴重𢳂頭人馬角力 > 400m,或中段窒步勒避需重新蹓躂),導致大敗。今仗有條件改善(如遇快步速幫助步韻,或除眼罩) → 實際體力未見底,被市場嚴重低估。
  4. `[姿態破格 The Dominator]`:近仗雖受惠於低消耗/靚位(1-2疊),但贏馬姿態呈統治級(輕鬆大勝/大把在手),證明其實力深不見底,超越同班對手,不應因走位舒服而受罰。
- **❌ Weak:**
  - `[實力見底 Exposed Illusion]`:近仗經歷「真低消耗」(慢步速配合+內欄黃金包廂/輕鬆單騎),且**走勢順暢無隱形消耗**,毫無物理蝕位卻依然大敗 → 證明其實力不足。
- **➖ Neutral:** 正常消耗對應正常賽果,或高消耗導致大敗(合乎物理常理)。

**疲勞累積效應:**
- 連續兩仗高消耗 = `疲勞中等`;連續三仗高消耗 = `疲勞嚴重`
- **→ 累積消耗達「中等」或以上 → 自動加入 Step 12 寬恕因素**

### Step 12: 寬恕檔案 (Forgive File)

| 寬恕因素 | 程度 |
|:---|:---|
| 三疊望空 | ⭐⭐⭐ 高度 |
| 受困/塞車 | ⭐⭐⭐ 高度 |
| 賽後不適(跛行/流鼻血) | ⭐⭐⭐ 完全 |
| 慢閘 | ⭐⭐ 中度 |
| 場地不利 | ⭐⭐ 中度 |
| EEM 累積消耗達「中等」或以上 | ⭐⭐ 中度 |
| 騎師失誤(一般) | ⭐ 有限 |
| 騎師失誤(競賽小組裁定) | ⭐⭐⭐ 完全 |

**規則:** ≥ 2 個寬恕因素 → 「上仗不可作準」→ **回溯 Step 2 重新錨定**

**[V 型反彈自動升級 (V-Rebound Auto-Upgrade)]:** 若 Step 12 結論為「上仗不可作準」,**且**觸發回溯後的可作準賽事顯示馬匹在「同場 + 同程」具備勝出/上名紀錄,**且**今仗回歸該首本場地/路程 → 此情境不僅是「數據重置」,更是**正面反彈訊號**。必須強制將 Step 2 情境標籤升級為 `[情境A-升級]`(而非僅中性錨定)。典型觸發:上仗跑非首本路程大敗 → 今仗回師勝途 + 好檔。

**⚠️ 反三重計算 (Anti-Triple-Count):** 若 V 型反彈已將情境標籤升級為 `[情境A-升級]`,則 Step 14.1 嘅寬恕輔助加分 (+1 ✅) **不再生效**。V 型反彈本身已包含數據重置 + 正面情境升級,屬於最強等級嘅寬恕受益,不可再疊加額外 ✅。擇一原則:V 型反彈 OR 輔助 ✅,不可兼得。

**[嚴重受困自動作廢 (Severe Blockage Auto-Void)]:** 若馬匹上仗名次 ≥10th,且評語明確出現「未能望空」/「直路受阻」/「無處展現勁勢」/「Held up」等受困描述,則該仗段速數據**完全失真**,此單一因素即足以觸發「上仗不可作準」,無需第二個寬恕因素。必須強制回溯至最近一仗「望空」的賽事重新錨定段速與 EEM。常見於谷草 B/C 賽道轉彎入直路時被困的情況。

**[競賽小組裁定自動作廢 (Stewards' Finding Auto-Void)]:** 若賽後評語或競賽小組報告顯示騎師被裁定騎法不當(如「未有盡力策騎」/「騎法受查」/「被競賽小組研訊」/「被罰停騎」/「策騎不合規格」等),則該仗賽果**完全不能反映馬匹真實能力**。此單一因素即足以觸發「上仗不可作準」,無需第二個寬恕因素。必須強制回溯至最近一仗「騎法正常」的賽事重新錨定段速與 EEM。此類賽事對該馬而言完全不具參考價值,不可作為評估其能力的依據。

**[SIP-HV6] [醫療事故自動作廢 (Medical Incident Auto-Void)]:** 若賽後評語或賽績短評明確記載馬匹出現醫療事故(如「流鼻血」/「跛行」/「不適」/「受傷」/「Bled」/「Lame」),則該仗賽果**完全不能反映馬匹真實能力**。此**單一因素**即足以觸發「上仗不可作準」,無需第二個寬恕因素。
- **穩定性維度影響:** 該仗結果(通常為包尾大敗)必須從穩定性計算中**完全排除**。例如:近6仗為 6/5/6/**12**/4/7,其中第4仗流鼻血 → 有效近績變為 6/5/6/4/7(5仗中入前5=4次=80%),穩定性從 ❌ 升至 ✅。
- **V 型反彈檢查:** 醫療事故後首仗復出,若配合同場同程回師+好檔 → 自動觸發 V 型反彈升級。
- **來源:** 2026-03-18 HV R3 萬眾開心:第4仗流鼻血跑第12名未被正確排除,導致穩定性被錯誤判定為 ❌,最終僅評 C+(實際跑第3名)。

## 📊 全段速剖面分析指引 (Full Sectional Profile Analysis Guide) [V3.0 — Phase 1 升級]

> [!CAUTION]
> **V3.0 強制升級:** Facts.md 包含 Python 預計算嘅全段速剖面表格同 EEM 預評估區塊。
> 分析師**必須逐段引用**呢啲預計算數據，**嚴禁只引用 L400 單一數字**。
> **違反此規則 = QG-CHECK 自動失敗。**

### 數據來源優先級
1. **Facts.md「📊 全段速剖面」表格** — 第一優先。包含各段實際時間 + Δ偏差（對比 HKJC 官方標準）
2. **Facts.md「⚡ EEM 預評估」區塊** — 第一優先。包含走位消耗、累積疲勞、觸發條件
3. **Facts.md「📐 頭馬距離趨勢」** — 第一優先。包含趨勢方向（收窄/擴大/波動）
4. **賽績表格中嘅 L400/L600** — 補充數據
5. **賽績短評** — 補充情境

### 全段速剖面表格解讀方法
- **S1, S2, S3...** = 各段實際完成時間（秒）
- **Δ1, Δ2, Δ3...** = 與 HKJC 官方班次標準嘅偏差。負數=比標準快，正數=比標準慢
- **形態** = 段速形狀分類（均速型/漸進加速/快開慢收/慢開快收/波動型）

### 強制分析維度（每匹馬必須完成所有 6 項）
1. **前段消耗:** Δ1/Δ2 為大負數 → 前段跑太快 → 後段衰退風險
2. **後段餘力:** 最後段 Δ 為負數 → 有餘力 → EEM 評分正面
3. **快開慢收:** 前段 Δ 大負數 + 後段 Δ 大正數 → 典型過度消耗形態
4. **漸進加速:** 各段 Δ 逐步變小或轉負 → 理想嘅能量分配
5. **形態一致性 [SIP-P1b]:** 近 3 仗形態是否穩定 → 反映馬匹跑法可預測性（見 Step 10.4）
6. **前段-後段因果 [SIP-P1c]:** S1 Δ 同 L400 Δ 嘅因果關係 → 判斷引擎質量（見 Step 10.5）

### 頭馬距離趨勢引用規範 [SIP-P1d]
- 分析師**必須**引用 Facts.md 嘅頭馬距離趨勢,格式: `頭馬距離趨勢: [📈收窄中/📉擴大中/📊波動] — [最近3仗數據]`
- 與段速趨勢進行交叉驗證（見 Step 10.6）
- 穩定性維度判定時必須參考頭馬距離趨勢（見 Step 14.1）

### EEM 預評估引用規範
- 分析師引用格式: `上仗走位(XW) → 消耗等級 → 累積疲勞 → 觸發條件`
- 若 EEM 觸發「逆境破格」→ 升級 EEM 潛力至 ✅
- 若 EEM 觸發「超級反彈候選」→ 重點標注，升級 EEM 潛力至 ✅
- 若 EEM 觸發「實力見底風險」→ 降級 EEM 潛力至 ❌
- 若 EEM 無觸發 → 根據消耗趨勢判定 ✅/➖/❌
