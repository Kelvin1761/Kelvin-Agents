# 場地邏輯模組:沙田草地 (Sha Tin Turf)

> 本模組僅適用於沙田草地賽事。
> 03-06 入面嘅沙田條件規則(Step 0 配置修正、Step 11 1000m/1200m EEM 特性、Step 14.2B 1200m 檔位豁免)自動生效。

## 配置對步速預測嘅修正 (注入 Step 0)
- **A Rail:** 維持原判
- **B/B+2:** 前領馬偷襲成功率上調。若原判為 Genuine,應向 Moderate 方向微調
- **C/C+3:** 前領馬偷襲成功率大幅上調。外檔後追馬機會顯著減少。龜速場景下外檔後追馬幾乎無機會

## 配置對 EEM 嘅修正 (注入 Step 11)
- **A Rail:** 走位消耗按標準表計算
- **C/C+3:** 3-wide 望空消耗**升一級**(中消耗 → 高消耗)。跑道窄意味同樣嘅外疊角度造成更大嘅幾何蝕位

## [SIP-ST40] 直路賽放頭壟斷覆蓋 (Straight Course Front-Runner Monopoly Override)

> [!CAUTION]
> **來源:** 2026-04-06 ST R1 飛來霸 (D 級, 23 倍勝出,全場唯一證明放頭馬,7 匹初出馬)。兩部引擎均走漏頭馬。

**觸發條件:**
```
IF 賽事距離 = 1000m 直路賽  
AND 場內具證明前速之放頭馬數量 = 1  
AND 初出馬佔比 > 40%  
THEN → 觸發「放頭壟斷覆蓋」
```

**定義「具證明前速」:** 近 5 仗 ≥2 仗 in-run 居前 3 位 且 L200 前速段時 ≤ 全場平均。

**效果:**
1. **Step 0 步速瀑布修正:** 該馬步速預測強制設為「無壓力自控 (Soft Lead / Crawl)」。`DOMINANCE_GAP = 明確 (Clear)`
2. **Step 14.1 新增輔助維度:** `⚡ 壟斷前速 [輔助]: ✅` — 自動授予,理據:「全場唯一證明前速之放頭馬」
3. **評級下限 = B-** — 無論歷史 L400 多慢,壟斷放頭馬之評級不可低於 B-
4. **SIP-ST6 龜速壟斷自動觸發:** 若引擎 = Type A → SIP-ST6 (a) 龜速壟斷微調自動滿足

**評語模板:**
```
> 📌 [壟斷覆蓋] 此馬為全場唯一證明前速之放頭馬,初出馬群體前速未知,
> 預計可無壓力帶放。歷史 L400 偏慢但放頭壟斷下不可作常規參考。
> 評級下限：B-
```

**不觸發:** 證明放頭馬 ≥2 匹(有前速競爭) 或 初出馬佔比 ≤40%(前速格局已知)
