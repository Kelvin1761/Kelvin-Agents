### Step 10: 段速法醫分析

**絕對禁止只看名次。**

> [!CAUTION]
> **[ANCHOR-段速] 全段速剖面強制引用 (Full Sectional Mandate):** 段速法醫分析**嚴禁只引用 L400 單一數字**。每匹馬必須引用 Facts.md「📊 全段速剖面」表格中嘅 **S1/S2/S3...各段時間** 及 **Δ1/Δ2/Δ3...偏差值**，對每段速度逐一解讀。若 Analyst 僅引用 L400 而忽略前段/中段數據 → **QG-CHECK 自動失敗**。

**10.1** 提取近仗全段速剖面（優先）及 L600/L400 段速
**10.2 段速修正(定性判斷,非精確計算):**

| 修正因素 | 條件 | 修正方向 |
|:---|:---|:---|
| 三疊望空 | 全程外疊無遮擋 | 實際能力 **優於** 末段時間所示 |
| 受困 | 直路/彎位受阻 | 實際能力 **明顯優於** 末段時間(未盡全力) |
| 逆偏差 | 內欄利但走外 | 實際能力 **優於** 名次所示 |
| 順偏差 | 享受利好偏差 | 實際能力 **弱於** 末段時間所示(打折扣) |
| 領放 | 前 400m 用力領放 | 末段放慢屬正常,不扣分 |

**⚠️ 保守修正原則:** 段速修正嘅結論係「實際能力優於/弱於時間所示」,屬**定性方向判斷**,不等於「大幅升級」。特別係「三疊望空」同時出現於段速修正 (Step 10) 同 形勢與走位 (Step 11) 時,段速修正應保守處理 — 僅確認「能力不弱於所示」,避免同 形勢與走位維度雙重膨脹。

**10.3** 近 3 仗修正後末段趨勢:逐仗改善 = `上升軌`;逐仗退步 = `衰退中`;持平 = `穩定`

**10.4 [ANCHOR-段速形態] 段速形態一致性分析 (Sectional Shape Consistency):**

> [!IMPORTANT]
> 從 Facts.md 全段速剖面中提取近 3 仗嘅「形態」分類（漸進加速/快開慢收/均速型/慢開快收/波動型），判斷形態一致性：

| 近 3 仗形態一致性 | 判定 | 意義 |
|:---|:---|:---|
| 3 仗相同形態 | `高一致性 ✅` | 跑法高度可預測,分析置信度高 |
| 2 仗相同 + 1 仗不同 | `中等一致性 ➖` | 基本可預測,但有變數 |
| 3 仗各不同 | `低一致性 ⚠️` | 跑法不穩定,段速數據參考價值降低 |

**形態一致性傳導:** 高一致性 → 段速法醫結論可信度加分;低一致性 → 段速法醫結論需降低權重,不可作為核心判定依據。

**10.5 [ANCHOR-段速因果] 前段-後段因果分析 (S1-to-L400 Causality):**

必須對每匹馬嘅近 3 仗,分析前段 Δ 同後段 Δ 嘅因果關係：

| 前段 Δ | 後段 Δ | 判讀 | 引擎質量訊號 |
|:---|:---|:---|:---|
| 大正數（慢開）| 大負數（快收）| 「儲力爆發」型 | 正常 — 需扣除儲力效應才能評估真實末段能力 |
| 大負數（快開）| 大正數（慢收）| 「過度消耗」型 | ⚠️ 負面 — 前段用力過度導致後段衰退 |
| 大負數（快開）| 大負數（快收）| 「全程強勢」型 | ✅✅ 極度正面 — 快開仍能快收 = 引擎質素極高 |
| 大正數（慢開）| 大正數（慢收）| 「全程不及班次」型 | ❌ 負面 — 即使儲力亦無餘勁 |

**量化門檻:** Δ 偏差 ≥ |0.5s| 為「大」偏差;Δ 偏差 < |0.3s| 為「輕微」偏差。

**10.6 [ANCHOR-頭馬距離] 頭馬距離趨勢驗證 (Margin Trend Verification):**

必須引用 Facts.md 嘅「頭馬距離趨勢」數據,同段速法醫結論進行交叉驗證：

| 段速趨勢 | 頭馬距離趨勢 | 交叉驗證結論 |
|:---|:---|:---|
| L400 上升軌 ✅ | 📈 收窄中 | ✅✅ 高度驗證 — 段速進步 + 距離收窄 = 真實進步 |
| L400 上升軌 ✅ | 📉 擴大中 | ⚠️ 矛盾信號 — 段速快但整體落後更多 = 可能係全場水平提升 |
| L400 衰退中 ❌ | 📈 收窄中 | ⚠️ 正面修正 — 段速字面退步但實際競爭力接近 = 班次因素 |
| L400 衰退中 ❌ | 📉 擴大中 | ❌❌ 高度負面 — 段速退步 + 距離拉大 = 確認衰退 |

**引擎類型段速解讀框架 (Engine Type Sectional Interpretation Framework):**

段速質量維度嘅 ✅/❌ 判定必須先確認引擎類型 (Step 2.6),再以該類型嘅預期段速模式作為比較基準:

| 引擎類型 | 預期段速模式 | L400 偏慢判定 | L400 偏快判定 |
|:---|:---|:---|:---|
| **Type A (前領均速)** | 均勻衰減,L400 不會特別快 | ➖ **正常** — 不應因此判 ❌ | ✅ **異常正面** — 前領型仍有末段加速 = 實力深不見底 |
| **Type B (末段爆發)** | L400 應為最快段,顯著快於全場平均 | ❌ **異常負面** — 標記 `⚠️ 引擎失效 (Engine Misfire)` | ✅ **正常** — 確認引擎正常運作 |
| **Type C (持續衝刺)** | 段速穩定,各段偏差小 | ➖ **正常** — 若整體段速穩定即可 | ✅ 若穩定中仍有加速 = 正面 |
| **Type A/B (混合)** | 無固定預期 | 按實際表現判斷 | 按實際表現判斷 |

**「引擎失效」(Engine Misfire) 規則:**
- 觸發條件:Type B 馬匹 + 近 3 仗 L400 持續排全場後半段(≥ 50% 分位)
- 效果:段速質量維度強制判為 ❌ Weak,且在分析中標記 `⚠️ 引擎失效 — 末段爆發型 L400 持續偏慢,核心武器失靈`
- 原理:Type B 嘅存在意義就係 L400 爆發力。若 L400 持續偏慢 = 引擎本身有問題,而不只係「一仗爬頺」

**→ 段速修正判斷傳入 Step 11 作 形勢 基準**

**10.7 [ANCHOR-走位段速複合] 走位-段速複合分析 (Position-Sectional Composite):**

> [!IMPORTANT]
> 段速質量必須結合走位消耗等級進行複合判讀。同一個 L400 數字,在高消耗走位下嘅含金量遠高於低消耗走位。

從 Facts.md 提取走位 (XW) 欄位嘅消耗等級,同段速 Δ 尾段偏差值進行交叉分析：

| 走位消耗等級 | 段速 Δ 尾段 | 複合判讀 | 段速質量修正 |
|:---|:---|:---|:---|
| 極高/高消耗 (≥3W) | 大負數（快於標準 ≥0.5s） | 「逆境段速」— 含金量極高 → 可觸發 SYN-01 | 段速 ✅✅（強確認） |
| 極高/高消耗 (≥3W) | 輕微偏差（±0.3s 內） | 「消耗抵消」— 實際能力優於字面 | 至少 ➖（不可判 ❌） |
| 極高/高消耗 (≥3W) | 大正數（慢於標準 ≥0.5s） | 「消耗壓垮」— 走位過差導致表現下降 | 按原判定 |
| 低/中低消耗 (≤2W) | 大負數（快於標準 ≥0.5s） | 「舒服快跑」— 段速快但需打折（走位太順） | ✅ 但降低權重 |
| 低/中低消耗 (≤2W) | 大正數（慢於標準 ≥0.5s） | 「實力見底」— 好位仍慢 → 可觸發 CON-01 | 確認 ❌ |

**走位來源:** Facts.md 賽績表「走位(XW)」+「消耗」欄位
**與互動矩陣嘅關聯:** 「逆境段速」自動觸發 SYN-01 判定;「實力見底」自動觸發 CON-01 判定

**10.7.2 賽事短評交叉驗證 (Running Commentary Cross-Validation):**

> [!IMPORTANT]
> Analyst 必須引用 Facts.md 賽績表「賽事短評」欄位,與走位/段速分析進行交叉驗證。

| 驗證項目 | 方法 | 觸發動作 |
|:---|:---|:---|
| Engine Type 驗證 | 若短評連續描述「居後列/留後」但 Engine = Type A（前領型） | → 標記 CONTRA：Engine 分類可能有誤,需重新評估 |
| 受阻模式識別 | 近 3 場均含「受擠迫/受阻/未能望空/被困」 | → 觸發寬恕加分考量（穩定性 ❌ 可能被外在因素扭曲） |

| 隱藏表現發掘 | 短評含「末段追近/衝刺有勁/被阻後追回」 | → 即使名次差,段速質量需重新審視（不可直判 ❌） |

### Step 11: 形勢與走位評估 (Draw & Positional Assessment)

> [!NOTE]
> 本步驟為**定性判斷**，只評估馬匹嘅檔位利弊、今日預計走位、近仗走位消耗、受阻/外疊/被困紀錄。**不得以步速作為此維度主要 ✅/❌ 理由。** 步速只保留於 race-level 戰場全景、段速質量、及最終 Top4 步速逆轉保險。

**場地幾何與走位豁免條款（必須優先檢查）:**
> [!IMPORTANT]
> 1. **1000m 直路賽:** 無彎道，外疊不構成幾何蝕位，此類賽事外疊走位風險**自動關閉**。
> 2. **末段抽出 (Late Sweep):** 前半程守 1-2 疊，僅最後 400m 抽出衝刺。正常最佳跑法，**不可**視為走位風險。
> 3. **泥地防沙機制 (AWT Kickback):** 沙田泥地 (AWT) 走外疊 **不一定** 不利。若為避開食沙而刻意走外疊走順步韻，反而較被困內欄食沙更優。
> 4. **沙田 1200m vs 1400m 起步點:** ST 1200m 排外檔 (10-14) 切入消耗極高；但 ST 1400m 有長直路緩衝，影響顯著較低。
> 5. **谷草 C/C+3 窄欄陷阱:** 跑馬地 C 或 C+3 跑道極度狹窄，排外檔 (10-12) 除非級數超班，否則極易陷入全程外疊劣勢。

**檔位與走位風險:**
- **[檔位數據優先]:** Facts.md 🎯檔位判讀為主要客觀錨點；需引用勝率/上名率/入Q率或同場同程檔位判讀。
- **[今日預計位置]:** 根據馬匹跑法、檔位、試閘/近仗出閘反應，判斷能否取得合理位置。
- **[近仗走位消耗]:** 優先引用近 3 仗 XW、沿途位、消耗等級、累積疲勞、受阻/外疊/被困等證據；若走位模式反覆、近 3 仗樣本不足、或有連續際遇問題，擴展至近 5 仗判斷趨勢。不可只用上仗單場定生死。
- **[初出馬替代]:** 初出馬用檔位數據 + 試閘跑法/出閘反應 + 預計初出適應風險；不得因無正式走位資料直接判 ❌。

**形勢與走位 ✅/❌ 判斷標準（定性）:**
- **✅ Strong:**
  1. Facts.md 🎯檔位判讀為 ✅有利，且馬匹跑法能自然取得合理位置。
  2. 檔位極利 + 前領/跟前或守內欄好位，減少轉彎/外疊消耗。
  3. 上仗走位蝕盡（受阻/外疊/被困）但今仗檔位/預計走位大幅改善。
  4. 上仗低消耗位置大勝或未盡全力，顯示位置優勢仍有餘量。
- **❌ Weak:**
  1. Facts.md 🎯檔位判讀為 ❌不利，且馬匹跑法需要蝕位或被迫後置。
  2. 急彎場地/窄直路 + 外檔 + 後追型，預計需要大外疊追上。
  3. 上仗靚位/低消耗順暢走位仍大敗，顯示位置紅利不足以補救能力缺口。
  4. 今仗檔位或預計走位明確惡化。
- **➖ Neutral:** 檔位/今日走位/近仗走位消耗均無極端正面或負面訊號

**走位蝕盡累積效應（定性）:**
- 連續 2-3 仗走位明顯不利（外疊/受困/慢閘）→ 視為寬恕因素，自動加入 Step 12 寬恕檔案

### Step 12: 寬恕檔案 (Forgive File)

| 寬恕因素 | 程度 |
|:---|:---|
| 三疊望空 | ⭐⭐⭐ 高度 |
| 受困/塞車 | ⭐⭐⭐ 高度 |
| 賽後不適(跛行/流鼻血) | ⭐⭐⭐ 完全 |
| 慢閘 | ⭐⭐ 中度 |
| 場地不利 | ⭐⭐ 中度 |
| 走位累積消耗達「中等」或以上 | ⭐⭐ 中度 |
| 騎師失誤(一般) | ⭐ 有限 |
| 騎師失誤(競賽小組裁定) | ⭐⭐⭐ 完全 |

**規則:** ≥ 2 個寬恕因素 → 「上仗不可作準」→ **回溯 Step 2 重新錨定**

**[V 型反彈自動升級 (V-Rebound Auto-Upgrade)]:** 若 Step 12 結論為「上仗不可作準」,**且**觸發回溯後的可作準賽事顯示馬匹在「同場 + 同程」具備勝出/上名紀錄,**且**今仗回歸該首本場地/路程 → 此情境不僅是「數據重置」,更是**正面反彈訊號**。必須強制將 Step 2 情境標籤升級為 `[情境A-升級]`(而非僅中性錨定)。典型觸發:上仗跑非首本路程大敗 → 今仗回師勝途 + 好檔。

**⚠️ 反三重計算 (Anti-Triple-Count):** 若 V 型反彈已將情境標籤升級為 `[情境A-升級]`,則 Step 14.1 嘅寬恕輔助加分 (+1 ✅) **不再生效**。V 型反彈本身已包含數據重置 + 正面情境升級,屬於最強等級嘅寬恕受益,不可再疊加額外 ✅。擇一原則:V 型反彈 OR 輔助 ✅,不可兼得。

**[嚴重受困自動作廢 (Severe Blockage Auto-Void)]:** 若馬匹上仗名次 ≥10th,且評語明確出現「未能望空」/「直路受阻」/「無處展現勁勢」/「Held up」等受困描述,則該仗段速數據**完全失真**,此單一因素即足以觸發「上仗不可作準」,無需第二個寬恕因素。必須強制回溯至最近一仗「望空」的賽事重新錨定段速與走位形勢。常見於谷草 B/C 賽道轉彎入直路時被困的情況。

**[競賽小組裁定自動作廢 (Stewards' Finding Auto-Void)]:** 若賽後評語或競賽小組報告顯示騎師被裁定騎法不當(如「未有盡力策騎」/「騎法受查」/「被競賽小組研訊」/「被罰停騎」/「策騎不合規格」等),則該仗賽果**完全不能反映馬匹真實能力**。此單一因素即足以觸發「上仗不可作準」,無需第二個寬恕因素。必須強制回溯至最近一仗「騎法正常」的賽事重新錨定段速與走位形勢。此類賽事對該馬而言完全不具參考價值,不可作為評估其能力的依據。

**[醫療事故自動作廢 (Medical Incident Auto-Void)]:** 若賽後評語或賽績短評明確記載馬匹出現醫療事故(如「流鼻血」/「跛行」/「不適」/「受傷」/「Bled」/「Lame」),則該仗賽果**完全不能反映馬匹真實能力**。此**單一因素**即足以觸發「上仗不可作準」,無需第二個寬恕因素。
- **穩定性維度影響:** 該仗結果(通常為包尾大敗)必須從穩定性計算中**完全排除**。例如:近6仗為 6/5/6/**12**/4/7,其中第4仗流鼻血 → 有效近績變為 6/5/6/4/7(5仗中入前5=4次=80%),穩定性從 ❌ 升至 ✅。
- **V 型反彈檢查:** 醫療事故後首仗復出,若配合同場同程回師+好檔 → 自動觸發 V 型反彈升級。
