## 8.2c 練馬師部署模式偵測 (Trainer Pattern Detection)

> **原理:** 除咗固定嘅條件表之外,以下通用模式可適用於所有練馬師,偵測到時標記為正面/負面訊號。

### 正面模式 (觸發 ✅ 或 ➖→✅)

| 模式 | 條件 | 標記 | 說明 |
|:---|:---|:---|:---|
| **騎師升級** | 上仗騎師排名低於今仗騎師(對比當季勝場排名)| `[騎師升級 ✅]` | 練馬師刻意配更好騎師 = 有信心出擊 |
| **轉廄即勝跟進** | 轉廄首仗勝出 + 今仗同練馬師第二仗 | `[轉廄跟進 ✅]` | 新練馬師調教見效 = 信心正面 |
| **放草刻意休息** | 休息 ≥28 天 + 上仗有走勢(前五名) | `[蓄意部署 ✅]` | 練馬師刻意讓有走勢馬匹放草恢復 = 目標明確 |
| **關鍵配備升級** | 首次戴眼罩/首去眼罩/首次戴面箍 + Tier 1 練馬師 | `[配備升級 ✅]` | 頂級練馬師嘅配備調整通常有針對性 |
| **首次轉場** | 從 ST 轉 HV(或反向)+ 相關距離有走勢 | `[轉場試驗]` | 中性至正面 — 練馬師認為場地轉換有利 |
| **連勝加碼** | 上仗勝出 + 今仗同場同距離 + 同騎師 | `[連勝鎖定 ✅]` | 全因素重複 = 練馬師認定配方有效 |

### 負面模式 (觸發 ❌ 或 ➖→❌)

| 模式 | 條件 | 標記 | 說明 |
|:---|:---|:---|:---|
| **騎師下降** | 上仗騎師排名高於今仗騎師(特別是 Tier 1 → Tier 3+)| `[騎師下降 ⚠️]` | 練馬師無法留住好騎師 = 信心存疑 |
| **頻繁換騎** | 近四仗配 4 位不同騎師 | `[找答案 ⚠️]` | 練馬師對此馬缺乏信心 |
| **升班無升級** | +2 以上升班 + 無騎師升級 + 無配備變動 | `[升班裸出 ⚠️]` | 無準備嘅升班 = 可能只為賺獎金 |
| **連續差仗同部署** | 近三仗同騎師+同走法+全差(≥7th) | `[策略疲勞 ⚠️]` | 練馬師無新方案 |

## 8.2e Step 8.2 綜合判定流程

每匹馬嘅練馬師訊號判定必須按以下流程:

1. **確認練馬師分級** (Tier 1/2/3)
2. **檢查場地偏好** — 今仗場地是否匹配
3. **掃描特定出擊訊號表** (8.2d) — 是否觸發任何已知模式
4. **掃描通用模式偵測** (8.2c) — 是否觸發正面/負面通用模式(騎師升級/下降、轉廄跟進、放草部署等)
5. **綜合判定:**
   - 觸發 ≥1 個特定訊號 OR ≥2 個正面通用模式 → `✅`
   - 觸發 1 個正面通用模式 + Tier 1 練馬師 → `✅`
   - 觸發 1 個正面通用模式 + Tier 2/3 → `➖` (中性偏正面)
   - 無任何觸發 → `➖`
   - 觸發 ≥1 個負面模式 → `❌`(除非同時觸發正面特定訊號可對沖)

6. **輸出格式:** 在分析中,`部署與練馬師訊號 (Step 8.2)` 需列明:
   - 練馬師分級 (Tier X)
   - 場地偏好匹配 (✅/❌/均衡)
   - 觸發嘅訊號 (標記名稱)
   - 騎師升級/下降判定
   - **轉倉因素 (若適用) [SIP-RR21]**
   - 最終訊號結論 (✅/➖/❌)

## 8.2f 轉倉因素 (Trainer Transfer Factor) [SIP-RR21]

> **原理:** 馬匹在練馬師之間轉倉後,因訓練方法轉變、環境刺激、部署策略改變等因素,首 1-3 仗可能出現「新鮮感紅利」,類似復出反彈效應。

### 數據識別
- 來源:排位表中「練馬師」欄位 vs 馬匹歷史資料「前練馬師」欄位
- 若兩者不同 → 標記為「轉倉馬」,並記錄轉倉後第幾仗

### 觸發條件與效果

| 場景 | 效果 | 維度歸屬 |
|:---|:---|:---|
| 轉倉首戰(新倉第1仗) | 練馬師訊號 **+0.5 輔助✅** | 輔助維度 |
| 轉倉後第2-3仗 | 練馬師訊號 **+0.25 輔助✅** | 輔助維度 |
| 轉倉後第4仗起 | 無加權(效應消退) | — |
| 低勝率→高勝率練馬師 (Tier 3→1/2) | 額外 **+0.25** | 練馬師訊號 |
| 高勝率→低勝率練馬師 (Tier 1→3) | **-0.25**(可能被放棄) | 練馬師訊號 |

### 練馬師勝率分級(用於轉倉方向判定)

| 月勝率 | 等級 | 對應 Tier |
|:---|:---|:---|
| ≥15% | 高勝率 | Tier 1 |
| 8-14% | 中勝率 | Tier 2 |
| <8% | 低勝率 | Tier 3 |

### 限制條件
- 轉倉因素為**輔助維度**,不可單獨改變評級等第
- 必須配合 ≥1 個其他正面訊號(好檔位/輕磅/強騎師)才生效
- 若馬匹有健康警示(流鼻血/足行/氣管問題)→ 轉倉因素不適用(轉倉可能因健康問題被放棄)

### 輸出格式
在 `部署與練馬師訊號 (Step 8.2)` 中呈現為:
`[轉倉首戰(蘇偉賢→廖康銘↑) +0.5輔助✅]` 或 `[轉倉第2仗(方嘉柏→蘇偉賢↓) -0.25]`

**來源:** 2026-03-22 ST R1 #10 綫路福將(蘇偉賢→廖康銘,轉倉首戰,D→亞軍 22x)
