# 馬匹分析骨架模板（Per-Horse Skeleton — HKJC Version V2）

> **用法:** Wong Choi 喺每個 Batch 開始前,將此骨架 × BATCH_SIZE 注入到 Analyst prompt。
> LLM 嘅任務係「填充骨架」而唔係「生成分析」,保證 100% 結構完整性。
> **核心邏輯/結論部分為 LLM 自由發揮區域** — 結構固定但分析深度同角度由 LLM 自由發揮。

> ⚠️ **P37 排位表事實錨點注入規則:**
> Wong Choi 喺注入骨架時,**必須**從排位表.md 提取以下數據並預填入 `📌 排位表事實錨點` 區域:
> - `賽績` string（原文照搬）
> - `評分` / `負磅` / `檔位` / `騎師` / `練馬師`
> - `上賽距今日數`
>
> **Analyst 嘅任務係「擴展解釋」呢啲已確認嘅事實，唔係自行從賽績重新提取。**
> **⛔ Analyst 嚴禁修改 `📌 排位表事實錨點` 區域嘅任何數據。**
