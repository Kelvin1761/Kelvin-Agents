# 即時評述句式模板庫 (梁浩賢風格)

所有模板用 `{H}` 代表馬名，`{N}` 代表檔位號碼，`{D}` 代表距離。

## 起步階段
```
- 「開閘啦！一起步個陣呀，{H}應聲彈出！」
- 「一起步個陣呀，{H}出閘慢左、擺左喺後面」
- 「{H}從{N}櫄出得好快呀」
```

## 中段
```
- 「到{D}米、帶出仍然係{H}」
- 「去到{D}米處，前面放頭既有{H1}、{H2}、{H3}，中間有{H4}、{H5}，後少少既馬仲有{H6}」
- 「{H}俾人夾住左！」
- 「{H}一路領放、走勢幾勁！」
```

## 入直路
```
- 「入正直路！後面馬開始變速！」
- 「前面放頭既仲有{H1}、{H2}、{H3}，中間{H4}、{H5}追左上黎呀」
- 「{H}喺外疊衝左上黎！」
- 「最後200米，領先緊既仲有{H}」
- 「{H}追緊上黎呀」
```

## 衝線
```
- 「衝線！跑第一名既馬{H}」
- 「衝線啦！{H}從後趕上勝出！」
- 「衝線！{H}從由頭帶到落尾勝出！」
- 「{H}贏到好似晨操咁跑番黎！」
- 「跑第二名既馬係{H2}」
- 「跑第三名既馬{H3}」
```

## 使用規則
1. Engine Dev 根據賽事狀態選擇合適嘅模板
2. 用馬匹實際名稱替換 `{H}` 佔位符
3. 每個階段隨機選 1-2 句，避免重複
4. 評述頻率：起步 1 句 → 中段每 500m 1 句 → 入直路 2-3 句 → 衝線 3 句
