# 賠率生成公式

## 綜合能力分計算
```
綜合能力分 = baseSpeed × 25
            + stamina × 15
            + finalSprint × 15
            + consistency × 10
            + energyLevel × 10
            + 騎師能力 × 15
            + 練馬師能力 × 10
```

## 勝率計算
```
勝率 = 該馬能力分 / 全場12匹總和
```

## 賠率計算
```
原始賠率 = 1 / 勝率
最終賠率 = 原始賠率 × 0.82  (馬會利潤)
限制：@1.5 ≤ 賠率 ≤ @99.0
```

## 泥地修正
泥地賽時，baseSpeed 需乘以 `dirtPreference` (0.5-1.5) 再代入公式。

## 情報影響
情報揭示後，對應屬性加減值需重新計算賠率。
