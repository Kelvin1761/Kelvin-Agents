# 數值平衡目標 (KPI)

## 賽事結果分佈目標

| 指標 | 目標 | 容許範圍 |
|:---|:---|:---|
| 熱門馬 (S/A) 勝率 | ~55% | 50-60% |
| 冷門馬 (D) 爆冷率 | ~5% | 3-8% |
| 領放馬勝率 (1200m以下) | ~35% | 30-40% |
| 後上馬勝率 (2000m以上) | ~30% | 25-35% |
| 泥地前置勝率 | ~40% | 35-45% |

## 經濟平衡目標

| 指標 | 目標 |
|:---|:---|
| 玩家 10 場後平均餘額 | $800-1200 |
| 玩家 50 場後平均餘額 | $500-800 |
| S 級達成率 (>$5000) | ~5% |
| F 級破產率 | ~15% |

## 驗證方法
所有 KPI 需通過 Monte Carlo 模擬 (≥1000 場) 驗證。
模擬假設：玩家每場投注 $100 獨贏熱門馬。
