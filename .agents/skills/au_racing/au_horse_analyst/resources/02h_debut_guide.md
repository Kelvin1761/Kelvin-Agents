# 初出馬及早期生涯馬分析指引 (AU Racing V1.0)

> **條件載入:** 當 WorkCard 標記 `career_tag = DEBUT / EARLY_CAREER / IMPORTED_DEBUT` 時載入此指引。
> `EARLY_CAREER` 馬匹使用標準流程，但必須參考本文件 Section B。

## 適用條件
- `career_tag = DEBUT` (Career = 0 場正式賽事)
- `career_tag = EARLY_CAREER` (Career = 1-5 場正式賽事)

## ⚠️ 重要：禁止錯誤分類
- Career ≤ 5 場嘅馬匹**嚴禁**被標記為「休出」或「First-up from spell」
- ✅ 正確表述：「馬匹初出」/「生涯第 X 場正式賽事」
- ❌ 錯誤表述：「馬匹休出後」/「休後復出」/「First-up from spell」

## A. DEBUT 馬 (Career = 0) — 7 維度特殊處理

> 使用同一套 7 維度矩陣，但每個維度嘅判定依據有所調整。
> **Rating Cap: A- 封頂。** 如果主要證據只係試閘，按 `02e_jockey_trainer.md` 更嚴格封頂 B/B+。

### 1. 狀態與穩定性 [核心]
判定依據轉為：**試閘質素**
- ✅ = 近 2 次試閘均勝出或跑入前 2
- ➖ = 試閘表現中規中矩 (上名但非前 2)
- ❌ = 近期試閘欠佳、未能上名、或試閘次數過少 (≤1次)

### 2. 段速與引擎 [核心]
判定依據轉為：**Sire 投影 + 試閘速度**
- ✅ = Sire AWD 吻合今仗距離 ± 200m，且試閘展現出符合 sire 特性嘅速度
- ➖ = Sire 數據不足 / AWD 與今仗距離偏差 > 200m / 試閘表現一般
- ❌ = Sire 為明確嘅長途/短途型而今仗距離完全唔吻合
- **必須引用** `03b/c/d_sire_*.md` 嘅 sire data 作為依據
- 若 sire 唔喺參考表中，按 `03a_sire_index.md` 嘅「未列種馬處理規則」搜索

### 3. 形勢與走位 [半核心]
正常判定（檔位/步速分析照常）

### 4. 騎練訊號 [半核心]
維持半核心，但判定指引更具體：
- ✅ = 一線騎師 + 大練馬師初出部署
- ➖ = 配搭正路
- ❌ = 二三線配搭

### 5. 級數優勢 [輔助]
- 初出馬 vs 初出馬 → 一律 ➖
- 初出馬 vs 有經驗馬（同場有 career ≥ 3 且有勝績嘅對手）→ ❌
- 例外：若初出馬嘅練馬師初出率特高 → 可維持 ➖

### 6. 場地適性 [輔助]
判定依據轉為：**Sire 場地統計**
- ✅ = Sire 子嗣喺今仗場地狀況 (Good/Soft/Heavy) 表現出色 (參考 `03b/c/d_sire_*.md`)
- ➖ = Sire 場地數據不足或表現中性
- ❌ = Sire 子嗣喺今仗場地狀況明確偏差 (e.g. Rubick 子嗣跑 Heavy)

### 7. 賽績線 [輔助]
- 初出馬一律判為 `➖` (無對手追蹤數據)
- **若試閘對手已正式出賽並有表現，可參考作為替代賽績線** (但信心度低)

### Rating Cap
- DEBUT 馬封頂 **A-**
- 即使矩陣分數高於 A-，亦不得超越
- trial-only / bloodline-only 不可直接衝 A-；通常 B 封頂，只有「試閘 + sire + trainer」三者同時強先可到 B+
- A- 只留畀明確 elite debut deployment（例如 Waller Metro Debut / Waterhouse-Bott Debut）而且 7 維判斷無重大 ❌
- DEBUT 馬不可升 S-tier；S-/S/S+ 需要正式賽事硬證據支持

## B. EARLY_CAREER 馬 (Career = 1-5) — 標準框架 + 注意事項

使用標準 7 維度判定，但分析時需注意：
1. 狀態週期標記為「生涯第 X 場」而非「X-up from spell」
2. **每場正式賽績都極珍貴** — 前仗、次前仗嘅數據全部要納入分析
   - 狀態與穩定性：即使只有 1-2 場賽績，都要完整分析走勢趨勢
   - 段速與引擎：即使只有 1-2 場 PI 數據，都應初步判定引擎類型
3. 賽績線只有 1-2 場對手追蹤時，每場追蹤結果都要詳盡分析
4. core_logic 必須提及「生涯經驗尚淺」作為風險因素
5. **Sire 數據作為輔助參考**：當正式賽績不足時，用 sire AWD/場地偏好補充引擎及場地維度判定
