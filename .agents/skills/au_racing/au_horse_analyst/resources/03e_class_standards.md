## 段速基準 (Sectional Benchmarks)

**Good 3 條件下的優秀段速參考（末段）：**

| 班次 | L600 | L400 | L200 |
|:---|:---|:---|:---|

| Group 2/3 | 33.5-34.0s | 22.3-22.7s | 11.1-11.3s |
| Listed / BM88+ | 34.0-34.5s | 22.5-23.0s | 11.2-11.5s |
| BM70-84 | 34.5-35.0s | 23.0-23.5s | 11.4-11.7s |
| BM58-66 | 35.0-35.5s | 23.5-24.0s | 11.6-12.0s |
| Maiden | 35.5-36.0s | 24.0-24.5s | 11.8-12.2s |

**判定規則：**
- L600 < 班次基準 = 優秀末段
- L200 ≤ 11.2s (高班) / ≤ 11.5s (中班) = 「有餘力」
- 衰退率 `(2×L200 - L400) / (L400 - L200)` > 15% = 體力問題
  > **公式說明：** L400 為末 400m 時間，L200 為末 200m 時間。前 200m split = L400 - L200，後 200m split = L200。衰退率 = (後半段 - 前半段) / 前半段 = (L200 - (L400 - L200)) / (L400 - L200) = (2×L200 - L400) / (L400 - L200)。正值代表後半段比前半段慢（體力衰退）。
