# Flemington 場地特性

| 項目 | 數據 |
|:---|:---|
| **周長** | 2312m |
| **直路** | 450m(極長) |
| **彎道** | 寬彎(pear-shaped 梨形佈局) |
| **方向** | 逆時針 |
| **場地** | 草地(大場,排水佳) |

## 關鍵特性
- **澳洲最長直路 (450m)** — 強烈利大步幅/後追型馬
- 梨形佈局 — 大彎道寬闊,外檔損耗比小場低
- **領放馬崩潰率極高 (60-70%)** — 長直路令前領優勢大幅削減
- 大場常分裂為 2-3 組,沿不同跑道側方競速
- 寬檔 (≥8) 在某些情況可能有利(接觸更新鮮外側草地)
- **風向追蹤器 (Wind Tracker)** — Flemington 設有即時風速監控,風向為關鍵變數

## 直線衝刺賽 (The Straight Six) — 1000m / 1100m / 1200m
> **交叉引用:** 若賽事為 Flemington 直線賽,必須同時載入 `04c_straight_sprint.md`

- **無彎道純速度測試** — 外檔處罰無效
- 場地狀態(乾/爛)同風向主導賽果
- 大場常分裂:一組靠內欄、一組靠看台側
- 爛地時外側草地通常更乾淨,靠外跑線較佳
- 首次跑直線馬匹統計上表現優於多次跑者
- 起步閘設於外欄側,確保均勻使用草地

## 場地偏差傾向

| 條件 | 偏差傾向 |
|:---|:---|
| 乾地 Good | 偏公平。後追型略有利(長直路) |
| Rail Out | 中性 — 長直路緩解 Rail Out 效應 |
| 濕地 | 外側草地可能更佳(特別直線賽) |
| 強逆風 | 前領崩潰率進一步上升 |
| 順風 | 前領存活率上升,但仍低於其他場 |
