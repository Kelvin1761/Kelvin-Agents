# Rosehill 場地特性

| 項目 | 數據 |
|:---|:---|
| **周長** | 2048m |
| **直路** | 408m |
| **彎道** | 較急(sweeping turns,近方形佈局) |
| **方向** | 順時針 |
| **場地** | 草地(平坦,排水中等) |

## 關鍵特性
- 近方形/三角形跑道佈局,急彎顯著
- 平坦場地,利高步頻/前領。大步幅外檔損耗大
- **Home Turn 急彎效應**:尾段急彎對後追外疊馬極度不利
- 濕地偏差常見 — Soft 6+ 時前領利好更明顯
- 乾地利 powerful gallopers,但急彎仍限制大步幅

> **交叉引用:** Rosehill 1200m 塞車連鎖效應 — 見 `02a-02g (split engine files)` 覆蓋規則「Rosehill 1200m Traffic Chain Reaction」

## 場地偏差傾向

| 條件 | 偏差傾向 |
|:---|:---|
| 乾地 Good | 偏利前置/箱位 |
| 乾地 + Rail Out | 強烈利前置 |
| 濕地 Soft/Heavy | 前置利好更明顯。需 Soft 實績 |
| 濕地 + Rail True | 內欄可能崩潰(Chopped Out),中外疊較佳 |
