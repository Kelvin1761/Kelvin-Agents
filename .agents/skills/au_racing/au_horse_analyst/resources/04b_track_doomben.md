# Doomben 場地特性

| 項目 | 數據 |
|:---|:---|
| **周長** | 1715m |
| **直路** | 350m |
| **彎道** | 中等偏急(5% banked) |
| **前後直路** | 各 320m(2% camber) |
| **方向** | 順時針 |
| **場地** | Kikuyu 草地(1996 翻新,排水系統極佳) |

## 關鍵特性
- 昆士蘭第二主場(與 Eagle Farm 同區,對面馬路)
- **類似 Moonee Valley 嘅急彎**,但直路較長 (350m vs 173m)
- 偏小場,利前領 + 前中段位置
- 1996 翻新後排水系統極佳,大雨後仍能維持場地
- **Brisbane Winter Carnival + Summer Carnival** 重要場地

## 場地偏差傾向

| 條件 | 偏差傾向 |
|:---|:---|
| 乾地 Good | 偏利前置 / 箱位 |
| 濕地 | 排水佳但仍偏利前置。濕地偏差常見 |
| Rail Out | 前置利好更顯著 |
