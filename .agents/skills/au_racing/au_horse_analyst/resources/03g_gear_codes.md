# 裝備代碼對照表 (Gear Code Reference)

> **用途:** Step 5 裝備解碼時,快速查找澳洲賽馬常用裝備代碼及其意義。
> 來源:Racing Australia 官方裝備分類。

## 常見裝備代碼

| 代碼 | 裝備名稱 | 用途/信號 |
|:---|:---|:---|
| **B** | Blinkers (眼罩) | 限制視野集中注意力。首戴=求變/搶放;除去=可能轉守 |
| **BO** | Blinkers Off | 除眼罩。常見轉至後上跑法 |
| **V** | Visor (半眼罩) | 比眼罩視野稍闊。用於需要集中但亦需看到旁邊馬匹的馬 |
| **W** | Winkers (側翼眼罩) | 輕度視野限制。常用於不需要全眼罩的馬匹 |
| **TT** | Tongue Tie (舌帶) | 固定舌頭防止氣道阻塞。首戴=改善呼吸/末段發揮 |
| **CP** | Cheek Pieces (面頰帶) | 輕度集中注意力。比眼罩溫和 |
| **NB** | Nose Band (鼻帶) | 控制頭部姿勢。常用於搶口/抬頭馬 |
| **XN** | Cross-over Noseband (交叉鼻帶) | 更強力拉低頭部。矯正抬頭問題 |
| **LB** | Lugging Bit (矯正銜鐵) | 矯正左右外閃。重要訊號:若配合頂級騎師=馬房已解決問題 |
| **P** | Pacifiers / Ear Muffs (耳塞) | 消音,減少起步前焦慮。常見「閘前除耳塞」(Ear Muffs Off) |
| **EMO** | Ear Muffs Off (閘前除耳塞) | 於起步欄除耳塞。精英馬房常用密碼(見 Waller Metro Debut) |
| **BP** | Bar Plates (蹄鐵板) | 保護受傷蹄部。首戴=蹄傷修復中,表現可能受限 |
| **GG** | Gelded (閹割) | 非裝備但常出現。首次閹後=可能改變性格,反彈訊號 |
| **BR** | Bubble Ring (鼻環) | 限制呼吸量,防止過度前衝 |
| **NC** | Nasal Strip (鼻撐) | 擴張鼻孔改善呼吸 |
| **TB** | Tongue Bit (舌銜鐵) | 控制舌頭位置 |

## 裝備變動訊號判讀

| 變動類型 | 信號強度 | 判讀 |
|:---|:---|:---|
| **首戴眼罩 (B first)** | 🟡 中 | 求變。搶放意圖或集中注意力 |
| **除眼罩 (BO)** | 🟡 中 | 可能轉跑法。配頂級騎師=正面 |
| **首戴舌帶 (TT first)** | 🟢 正面 | 改善呼吸/末段發揮。精英馬房成功率高 |
| **首戴矯正銜鐵 (LB first)** | 🟢 正面 | 針對性修正已知問題 |
| **首戴面頰帶 (CP first)** | 🟡 中 | 溫和調整,效果未知 |
| **首戴蹄鐵板 (BP first)** | 🔴 負面 | 蹄傷修復中,體能可能受限 |
| **閹後首出 (GG first)** | 🟢 正面 | 性格改變,常見反彈。特別是年輕遲熟馬 |
| **≥2 項同時首戴** | 🟡⚠️ | 大量變動=馬房全力出擊 OR 無計可施。需配合練馬師訊號判斷 |

> **精英馬房裝備升權規則(見 `02_algorithmic_engine.md` Step 5):** Waller / Snowden / Freedman / Hawkes / Maher / Neasham 的首戴裝備成功率顯著高於平均。
