# 場地解碼器 — 條件讀取框架

> **[條件讀取協議]** 本檔案為場地分析嘅通用框架。具體賽場特性、環境邏輯按 Wong Choi 提供嘅路由標籤條件載入：
>
> - `[TRACK_MODULE]` → 載入對應嘅 `04b_track_[venue].md`（只載入當日賽場）
> - `[GOING: SOFT_5+]` → 載入 `04d_wet_track.md`（濕地邏輯）
> - `[SURFACE: SYNTHETIC]` → 載入 `04e_synthetic.md`（膠沙地邏輯）
> - `[RACE_TYPE: STRAIGHT_SPRINT]` → 載入 `04c_straight_sprint.md`（直線衝刺模組）

## 通用場地分析原則

1. **場地適性判斷**：該場贏馬紀錄 → +15%。首次轉向+急彎 = Risk High
2. **州際風險**：首次跨州 -5%，東部→西澳 -10%
3. **步幅匹配**：大步幅 = 長直路佳/急彎損 | 高步頻 = 小場佳
4. **場地數據格式**：每個場地檔案包含周長、直路長度、彎道特性、方向、關鍵特性
