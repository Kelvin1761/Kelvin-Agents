# Eagle Farm 場地特性

| 項目 | 數據 |
|:---|:---|
| **周長** | 2027m |
| **直路** | 434m(長直路) |
| **彎道** | 寬彎(gradual sweeping turns) |
| **方向** | 順時針 |
| **場地** | Kikuyu 草地 + 沙底木纖維底層(woodfibre/sand composite) |

## 關鍵特性
- **昆士蘭主場** — 大場、長直路、寬彎
- 「Galloper's Track」— 利大步幅/後追型馬,獎勵真正嘅 class 同耐力
- 兩個大寬彎讓馬匹可穩定節奏
- **場地歷史問題**:Kikuyu 草嘅 thatch 層曾導致表面積水 — 無雨仍可能跑 Heavy
- 近年翻修後場地改善,Ryegrass 補種
- **Brisbane Winter Carnival (5-6 月)** 核心場地:Stradbroke Handicap、QLD Derby、JJ Atkins

## 場地偏差傾向

| 條件 | 偏差傾向 |
|:---|:---|
| 乾地 Good | 偏公平。長直路利強後追 |
| Rail Out | 前置微有利,但長直路仍助後追 |
| 濕地 | 場地處理雨水能力中等(歷史 thatch 問題)。需 Soft 實績 |
| 大雨後 | 即使官方偏好場地,thatch 問題可能令實際更重 |
