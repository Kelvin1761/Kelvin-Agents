# Randwick 場地特性

| 項目 | 數據 |
|:---|:---|
| **周長** | 2224m |
| **直路** | 410m(含上坡段 "The Rise") |
| **彎道** | 寬彎(sweeping bends) |
| **方向** | 順時針 |
| **場地** | 草地(排水一般,小雨即有影響) |

## 關鍵特性
- **「The Rise」上坡段**:終點前 300m 有一段明顯上坡,測試馬匹耐力與加速力。強後追型馬受益
- 大場、寬彎,利大步幅馬充分伸展
- 排水能力一般 — 微量降雨即可能令場地偏爛,影響跑法
- **風向影響**:強南風時有利前領馬(背風跑彎道側直路段)

## 場地偏差傾向

| Rail Position | 偏差傾向 |
|:---|:---|
| True | 中性偏公平。前領勝率 ~31%,後追 ~29% |
| Rail Out (+3 - +5m) | 偏利前置 / 貼欄 |
| Rail Out (+6m+) | 強烈利前置。後追馬機會顯著減少 |
| 濕地 + Rail Out | 最內欄可能反而成「慢線」,中外疊反而較快 |
